## Рабочие процессы

### Начало сессии

1. `git pull` для синхронизации
2. Проверить `000_Inbox/` на новые элементы

### Управление проектами

- Все материалы проекта в соответствующей папке `200_Projects/`

### Контент-процесс (AISobolev)

- Идеи → `310_Content_Ideas/`

- Черновики → `350_Draft_Posts/`
- Опубликовано → `360_Published_Posts/`
