- **Почему важно:** Это ключевой переход от ремесленника к предпринимателю. Платформа AGIents позволяет делегировать без дорогих разработчиков.
- **Следующие шаги:**

  - Оценить результат работы Дениса Сорокина с демо-проектом
  - Освободившееся время → на продажи

### Тема 2: Продукт = база знаний + агенты

- **Где проявилась:** Фичи AGIents (импорт/экспорт архивом, тройка агентов для авто-генерации БЗ), пост "База знаний — первый шаг", загрузка БЗ по стоматологии, ТЗ по санаториям
- **Суть:** На этой неделе окончательно выкристаллизовалась core-ценность AGIents: не просто "создай бота", а полный цикл работы с базой знаний — загрузка сырых данных → автоматическая обработка тройкой агентов (парсинг, категоризация, дедупликация) → структурированная чистая БЗ → агент-консультант.
- **Почему важно:** Это конкурентное преимущество. Большинство конкурентов дают "закинь документ и спрашивай". AGIents делает **качественную** базу знаний автоматически.
- **Техническая заметка:** Есть known issue с race condition при дедупликации (асинхронная обработка). Решение: семафор на уровне папки или пост-обработка.
- **Следующие шаги:**
  - Пофиксить дедупликацию (race condition)
  - Протестировать на клиентских данных стоматологии
  - Использовать этот flow как демо для новых клиентов
