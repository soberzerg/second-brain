## Контекст

Mediashop24 — интернет-магазин с 15-летней историей, работающий с премиальным немецким брендом WMF (посуда, кухонная техника, аксессуары). ~100 SKU, основные — менее 50.

**Текущие каналы продаж:**
- **Ozon** — основной канал, ~90% выручки
- Wildberries — в стадии раскачки
- Яндекс.Маркет — есть, но мало заказов
- Собственный сайт (PrestaShop) — заброшен, SEO убит
- Telegram-канал — только начинает развиваться

**Ключевые боли (по итогам встречи):**
1. Ручная аналитика Ozon занимает много времени, закономерности находятся медленно
2. Telegram-канал пустой, нет ресурса на регулярное наполнение контентом

## Направление 1: ИИ-аналитик Ozon

### Проблема

Федор ежедневно вручную:
1. Заходит в Ozon Seller, выгружает отчёты по каждому SKU
2. Переносит данные в Google Sheets (отдельная вкладка на каждый SKU)
3. Записывает свои действия (изменил ставку, снизил цену)
4. Глазами ищет закономерности между действиями и результатами
5. Формирует гипотезы и проверяет их

**Проблемы текущего подхода:**
- Занимает ~1-1.5 часа в день
- Разные SKU реагируют по-разному на одинаковые действия — закономерности не очевидны
- Нет системной памяти: какие гипотезы уже проверялись, что сработало, что нет
- Пробовали Comet (Perplexity) — не сохраняет данные между сессиями, неудобно

### Решение

Автоматизированная система аналитики, которая:

**Этап 1 — Сбор данных (автоматизация рутины)**
- Автоматическая выгрузка данных из Ozon Seller API (или парсинг, если API ограничен)
- Запись в структурированную базу данных (не Google Sheets)
- Ежедневное обновление без участия человека

**Этап 2 — Анализ и рекомендации (ИИ-аналитик)**
- Ежедневный отчёт: что изменилось за сутки, аномалии, тренды
- Корреляционный анализ: как действия (изменение ставки, цены) влияют на метрики (показы, CTR, конверсии, продажи) по каждому SKU
- Кластеризация SKU: группировка товаров по типу реакции на изменения (вместо "каждый SKU уникален" — выявление паттернов)
- Рекомендации: "На SKU X просели показы на 15%, исторически для этого кластера помогало снижение цены на 50-100р при сохранении ставки"
- Мониторинг конкурентов: отслеживание цен ключевых конкурентов по WMF на Ozon

**Этап 3 — Обучение на решениях (память)**
- Федор принимает решение, отмечает его в системе
- Через 3-7 дней система оценивает результат
- Накапливается база "действие → результат" по каждому SKU и кластеру
- Со временем рекомендации становятся точнее

### Техническая реализация

```
Ozon Seller API
      ↓
n8n (оркестрация, расписание, триггеры)
      ↓
PostgreSQL / Supabase (хранение данных)
      ↓
Claude API / Gemini API (анализ, рекомендации)
      ↓
Telegram-бот (отчёты Федору) + Google Sheets (для привычного просмотра)
```

**Стек:**
- **n8n** — оркестрация, расписание выгрузок, триггеры на аномалии
- **PostgreSQL** (или Supabase) — хранение исторических данных, действий, результатов
- **LLM API** (Claude / Gemini) — анализ данных, генерация гипотез, рекомендации
- **Telegram-бот** — ежедневные отчёты и алерты Федору
- **Google Sheets** (опционально) — экспорт для привычного просмотра

### Что получает клиент

| Было | Стало |
|---|---|
| 1-1.5 часа/день на ручной сбор данных | Данные собираются автоматически |

| Гипотезы в голове, забываются | Все решения и результаты сохраняются |
| "Тыканье пальцем в небо" на новых SKU | Рекомендации на основе похожих SKU |
| Нет мониторинга конкурентов | Автоматическое отслеживание цен |

## Направление 2: Контент-бот для Telegram

### Проблема

Telegram-канал mediashop24 только начинает развиваться. Контента мало, делать его вручную — нет ресурса. Федор видел, как через n8n люди делают автопостинг новостей, хочет подобное.

### Решение

Автоматический контент-конвейер для Telegram-канала:

**Модуль 1 — Мониторинг и отбор**
- Парсинг источников: кулинарные блоги, новости про кухонную технику, WMF-новинки, тренды посуды
- Мониторинг зарубежных источников (WMF Germany, кулинарные медиа)
- Мониторинг конкурентов в Telegram (каналы про посуду, кухню, товары для дома)
- Фильтрация по релевантности через LLM

**Модуль 2 — Генерация контента**
- Типы постов:
  - **Новости:** новинки WMF, тренды кухонной техники
  - **Полезный контент:** советы по уходу за посудой, лайфхаки (как у них в блоге, но адаптировано под Telegram-формат)
  - **Продуктовые посты:** обзоры конкретных товаров из каталога с ссылкой на Ozon/сайт
  - **Вовлечение:** опросы, викторины, сравнения ("WMF Devil vs WMF Profi — что выбрать?")
- Адаптация стиля под tone of voice канала
- Генерация изображений / подбор из каталога

**Модуль 3 — Публикация**
- Автопостинг по расписанию (3-5 постов в неделю)
- Драфты отправляются Федору на аппрув в личку бота
- Федор отвечает "ОК" — пост публикуется. Можно отредактировать или отклонить
- Аналитика: какие посты получают больше просмотров/реакций

### Техническая реализация

```
RSS / парсеры / API источников
          ↓
n8n (оркестрация, расписание)
          ↓
Claude API / Gemini API (фильтрация, генерация текстов)
          ↓
Telegram Bot API
    ↓              ↓
Федору на аппрув   → Канал @mediashop24
```

**Стек:**
- **n8n** — оркестрация, парсинг RSS, расписание
- **LLM API** (Claude / Gemini) — генерация и адаптация контента
- **Telegram Bot API** — аппрув + публикация
- **Firecrawl / RSS** — парсинг источников

### Что получает клиент

| Было | Стало |
|---|---|
| Канал пустой, постят от случая к случаю | 3-5 постов в неделю на автомате |
| Нет идей для контента | Система сама находит релевантные темы |
| Ручной поиск и написание | Федор только аппрувит готовые драфты |
| Нет аналитики | Видно, какие темы заходят аудитории |

### Этапы и сроки

| **v1.0** | Мониторинг конкурентов, автоподбор изображений, A/B тестирование форматов | +1-2 недели |

## Рекомендация: с чего начать

**Рекомендую начать с Направления 1 (ИИ-аналитик Ozon):**
- Это закрывает главную ежедневную боль Федора
- Результат измерим: экономия времени + качество решений
- Ozon — 90% выручки, улучшения здесь дают прямой эффект на деньги
- Пилот за $500 позволяет проверить ценность за 2 недели

**Направление 2 (контент-бот) можно запустить параллельно или после** — оно проще в реализации, но эффект на выручку не прямой и зависит от роста канала.
