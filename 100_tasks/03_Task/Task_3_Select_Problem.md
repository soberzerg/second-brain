## ВЫБРАННАЯ ПРОБЛЕМА

### Формулировка

> **"Компании не могут быстро и доступно создать кастомного ИИ-агента для автоматизации своих процессов."**

### Детализация

| Аспект | Описание |
|--------|----------|
| **Кто** | B2B компании (МСБ и Enterprise) в России/СНГ |
| **Что** | Хотят ИИ-агента для автоматизации (support, sales, operations) |
| **Барьер** | Нет доступного инструмента: разработка дорогая (100-500K RUB), долгая (3+ недель), требует экспертизы |
| **Боль** | Теряют деньги на ручной работе, упускают клиентов, отстают от конкурентов |
| **Размер** | Тысячи компаний в RU/CIS готовы платить 30-50K RUB/мес |

### "€8,000 момент"

**Конкретная измеримая боль:**

1. **Потерянные инвестиции:** Клиент заказывает разработку агента за 200-500K RUB, через 3 месяца понимает, что это не то, что нужно

2. **Упущенная выручка (Amaks):** 40% потерянных клиентов × средний чек 30K RUB × 365 дней = миллионы рублей в год

3. **Время:** 10+ часов в неделю на ручную работу вместо автоматизации

## Решение: AGIents.pro

### Продуктовая гипотеза

**AGIents.pro — no-code платформа для создания ИИ-агентов:**

| Компонент | Решает проблему |
|-----------|-----------------|
| **No-code builder** | #1: Быстрое создание без разработчиков |
| **Templates library** | #2: Не начинать с нуля |
| **Knowledge Base Builder** | #3: Помощь в подготовке данных для RAG |
| **Self-hosted option** | RU рынок: data sovereignty |
| **Hospitality template** | #5: Готовое решение для санаториев/отелей |

### Positioning

> "Единая no-code платформа для создания ИИ-агентов. Запустите своего агента за часы, не месяцы."

### Конкурентное преимущество

| Фактор | AGIents.pro | Конкуренты (Stack AI, Relevance AI) |
|--------|-------------|-------------------------------------|
| Локализация RU | ✅ | ❌ |
| Self-hosted | ✅ | Только Enterprise ($$$) |
| Цена | 30-50K RUB/мес | $500-2000/мес |
| Templates для RU | ✅ (Amaks как proof) | ❌ |
| Поддержка на русском | ✅ | ❌ |
