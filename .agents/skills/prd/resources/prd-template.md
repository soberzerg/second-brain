## Правила заполнения

- Заполняй **все секции** на основе собранных ответов
- Если информации по секции не было - напиши рекомендацию на основе контекста
- Технический стек: учитывай предпочтения из CLAUDE.md (Go, PostgreSQL, React+Vite+Tailwind)
- Этапы разработки: разбивай реалистично, MVP - минимально возможный
- Вызовы: будь честен, не скрывай сложности
