# Гайд по написанию хуков

**Выжимка из лучших источников по копирайтингу**

---

## Почему хук — это 80% успеха поста

> "В среднем, заголовок читают в 5 раз больше людей, чем основной текст. Когда вы написали заголовок, вы потратили 80 центов из своего доллара."
> — David Ogilvy

В Telegram/Threads/LinkedIn первые 2-3 строки решают всё. Читатель за 2 секунды решает: скроллить дальше или остаться.

---

## 7 типов хуков (Great Leads Framework + LinkedIn Analysis 2026)

### 1. Story Lead (История)

**Суть:** Начинаешь с конкретной истории, которая втягивает читателя.

**Формула:**

```
[Конкретная деталь/момент] + [Неожиданный поворот или интрига]
```

**Примеры:**

- "У меня было $47 на счету, когда я подписал первого клиента."
- "Нанял ассистента. Первая неделя: объясняю каждую мелочь, трачу больше времени чем экономлю."
- "Вчера закрыл месяц. Выручка больше, чем моя зарплата в найме. За 3 месяца."

**Когда использовать:**

- Есть реальный кейс или личный опыт
- Хочешь создать эмоциональную связь
- Сложная тема, которую легче объяснить через историю

**Сила:** Истории обходят "рекламный фильтр" мозга. Люди запрограммированы слушать истории.

---

### 2. Secret Lead (Секрет)

**Суть:** Обещаешь раскрыть что-то скрытое, малоизвестное, инсайдерское.

**Формула:**

```
[Большинство не знает/делает неправильно] + [Я покажу как на самом деле]
```

**Примеры:**

- "90% начинающих фаундеров тратят первый месяц на логотип вместо продаж. Вот почему."
- "Есть один приём в холодных сообщениях, который утроил мою конверсию."
- "Инвесторы никогда не скажут вам это в лицо: вам не нужны их деньги."

**Когда использовать:**

- Есть контринтуитивный инсайт
- Знаешь что-то из практики, чего нет в "официальных" источниках
- Хочешь позиционироваться как инсайдер

**Осторожно:** Не превращай в кликбейт. Секрет должен быть реальным.

---

### 3. Problem Lead (Проблема)

**Суть:** Называешь боль читателя так точно, что он думает "это про меня".

**Формула:**

```
[Конкретная боль + узнаваемые детали] + [Я тоже через это прошёл / Есть решение]
```

**Примеры:**

- "3 месяца готовишься к запуску. Клиентов — ноль. Знакомо?"
- "Каждый вечер открываешь ноутбук поработать над своим проектом. И каждый вечер залипаешь в чужой контент."
- "Читаешь про чужие успехи, а сам не можешь найти первого клиента."

**Когда использовать:**

- Точно знаешь боль своей аудитории
- Есть решение, которое реально помогает
- Хочешь высокую конверсию в "дочитал до конца"

**Сила:** Люди больше мотивированы избежать боли, чем получить удовольствие (loss aversion).

---

### 4. Promise Lead (Обещание)

**Суть:** Прямое обещание конкретного результата.

**Формула:**

```
[Конкретный результат] + [За конкретное время / конкретным способом]
```

**Примеры:**

- "Как я получил 5 клиентов за 2 недели без единого рубля на рекламу."
- "3 проекта параллельно, $2000/мес. Без офиса и без сотрудников."
- "От нуля до первого платящего клиента за 14 дней. Точный план."

**Когда использовать:**

- Есть измеримый результат
- Можешь доказать обещание (кейс, скриншоты, цифры)
- Аудитория уже "тёплая" и доверяет тебе

**Осторожно:** Обещание должно быть правдивым и достижимым. Иначе — потеря доверия.

---

### 5. Proclamation Lead (Провокация/Заявление)

**Суть:** Смелое, возможно спорное утверждение, которое вызывает реакцию.

**Формула:**

```
[Контринтуитивное/смелое заявление] + [Пауза или сразу аргумент]
```

**Примеры:**

- "Вашему бизнесу не нужен сайт. Ему нужен первый клиент."
- "Бизнес-план — самая бесполезная вещь, которую я написал."
- "Консалтинг — лучший старт для IT-бизнеса. И худший способ масштабировать."
- "Вам не нужен ко-фаундер. Вам нужна дисциплина."

**Когда использовать:**

- Есть сильная позиция, которую можешь аргументировать
- Готов к дискуссии в комментах
- Хочешь выделиться из потока "безопасного" контента

**Сила:** Провокация — это поляризация. Часть отпишется, но оставшиеся станут лояльнее.

---

### 6. Preview Lead (Превью/Анонс)

**Суть:** Даёшь "трейлер" того, что будет в посте.

**Формула:**

```
[Что ты узнаешь/получишь] + [Почему это важно именно сейчас]
```

**Примеры:**

- "Мой полный стек для соло-IT-бизнеса. С ценами и альтернативами."
- "Сегодня покажу, как нашёл первых 3 клиентов без бюджета. Будут шаблоны и грабли."
- "3 инструмента, которые заменяют мне команду из 5 человек. Без воды, только практика."

**Когда использовать:**

- Есть структурированный полезный контент
- Аудитория уже знает тебя и ждёт пользы
- Хочешь установить ожидания

**Слабость:** Наименее "цепляющий" тип. Работает только с лояльной аудиторией.

---

### 7. Contrarian Thesis (Контр-тезис)

**Суть:** Первая строка — мини-тезис, который переворачивает общепринятое мнение. Не кликбейт, а позиция.

**Формула:**

```
[Общепринятое ожидание] + [Переворот/отрицание] + [Интрига "почему"]
```

**Примеры из топовых LinkedIn-постов (февраль 2026):**

- "You don't need a co-founder. You need discipline and one paying client." (founder take)
- "The best marketing for a bootstrapped startup is cold DMs. Not ads. Not SEO. Not content." (contrarian tactic)
- "Consulting is the best way to start a business. And the worst way to scale one." (founder paradox)
- "Stop building. Start selling. Your MVP is a Google Doc and a Stripe link." (provocative reframe)

**Когда использовать:**

- Есть позиция, подкреплённая данными или личным опытом
- Тема, по которой все думают одинаково (и ты считаешь иначе)
- Хочешь максимальный reach (contrarian = самый мощный формат на LinkedIn)

**Почему работает лучше других хуков на LinkedIn:**

- Создаёт когнитивный диссонанс — читатель не может не продолжить
- Позиционирует автора как думающего независимо, а не повторяющего хайп
- Anti-hype работает лучше хайпа (Shi: "only 15% solvable", Rashidi: "AI was wrong")

**Разница с Proclamation Lead:** Proclamation — это смелое заявление ("Prompt engineering умрёт"). Contrarian Thesis — это переворот конкретного общепринятого мнения с обещанием объяснить почему. Proclamation шокирует. Contrarian Thesis заставляет пересмотреть.

## Принцип Data + Narrative + Framework

> **Из анализа 8 топовых LinkedIn-постов (февраль 2026)**

Самые успешные посты объединяют три слоя:

**1. Data** — конкретные цифры в первых строках:

- Не "много клиентов", а "47 холодных сообщений, 3 клиента"
- Не "хорошая выручка", а "$2000 MRR за 3 месяца"
- Не "быстро окупился", а "первый клиент на 14-й день"

**2. Narrative** — одна конкретная история или момент:

- Момент, когда первый клиент заплатил
- История провала с первым наймом
- "Вчера закрыл месяц — выручка больше зарплаты в найме"

**3. Framework** — модель/структура для понимания:

- → стрелки для списков
- "Раньше/Сейчас" для контраста
- Numbered lists для шагов
- Visual separators (++++++) для секций

**Комбинации и их эффект:**

- Все три = максимальный reach
- Только цифры = сухо, нет эмоции
- Только история = нет доверия, кажется анекдотом
- Только фреймворк = абстрактно, не запоминается

## Принцип "Slippery Slide" (Sugarman)

> "Каждый элемент текста имеет одну цель — заставить прочитать следующий элемент."

**Как применять:**

1. **Первая строка** → заставляет прочитать вторую
2. **Вторая строка** → заставляет прочитать третью
3. И так далее...

**Техники:**

- **Незаконченная мысль:** "И тут я понял одну вещь..."
- **Открытый вопрос:** "Но почему это работает?"
- **Противоречие:** "Звучит просто. На практике — ад."
- **Конкретика после абстракции:** "Это изменило всё. Вот как."

**Плохо:**

```
Сегодня расскажу про автоматизацию. Автоматизация важна для бизнеса.
Она экономит время и деньги.
```

**Хорошо:**

```
Я тратил 4 часа в день на рутину.
Теперь — 20 минут.
Что изменилось?
```

## 5 уровней осведомлённости аудитории (Schwartz)

Хук должен соответствовать тому, насколько аудитория знакома с проблемой и решением.

### Уровень 1: Unaware (Не осознают проблему)

**Аудитория:** Не знают, что у них есть проблема.

**Стратегия:** Начни с истории или провокации. Не продавай — заинтересуй.

**Пример:**

```
❌ "Запустите IT-бизнес без инвестиций"
✅ "Мой знакомый ушёл из найма и через 3 месяца зарабатывает больше. Без единого рубля стартового капитала."
```

### Уровень 2: Problem-Aware (Осознают проблему)

**Аудитория:** Знают, что что-то не так, но не знают решения.

**Стратегия:** Назови боль точно. Покажи, что понимаешь.

**Пример:**

```
❌ "Начните свой бизнес"
✅ "Вы уже полгода мечтаете уйти из найма. Каждый понедельник — одно и то же. Это можно изменить."
```

### Уровень 3: Solution-Aware (Знают о решениях)

**Аудитория:** Знают, что есть решения (автоматизация, боты), но не знают о твоём.

**Стратегия:** Покажи, чем твоё решение лучше/уникальнее.

**Пример:**

```
❌ "Запустите бизнес без денег"
✅ "Консалтинг → продуктизированная услуга → SaaS. Каждый шаг финансирует следующий. $0 стартового капитала."
```

### Уровень 4: Product-Aware (Знают о твоём продукте)

**Аудитория:** Слышали о тебе, но не купили/не попробовали.

**Стратегия:** Сними возражения. Покажи результаты других.

**Пример:**

```
❌ "У меня есть блог про бизнес"
✅ "3 подписчика применили мою схему холодных DM. Один закрыл клиента за неделю. Вот что он сделал иначе."
```

### Уровень 5: Most Aware (Готовы к действию)

**Аудитория:** Знают тебя, доверяют, готовы действовать.

**Стратегия:** Прямой оффер. Без долгих прелюдий.

**Пример:**

```
"Открываю менторство для 3 фаундеров. $0 → первый клиент за 30 дней. Кто первый — пишите в личку."
```

## Фреймворк 1-2-3-4 (Ship 30 for 30)

Структура первых 4 строк для максимального захвата внимания:

```
1. [Хук — зацепка]
2. [Усиление — развитие интриги]
3. [Контекст — почему это важно]
4. [Переход — мост к основному контенту]
```

**Пример для поста про бутстрэп:**

```
1. Я запустил бизнес с $0 на счету.                      ← Хук (WTF?)
2. Без инвестиций, без связей, без бизнес-образования.    ← Усиление (противоречие)
3. Через 6 месяцев — 3 проекта и стабильный доход.       ← Контекст (интрига)
4. Вот 3 вещи, которые я сделал иначе:                   ← Переход
```

## Чеклист хорошего хука

### Перед публикацией проверь:

- [ ] **Конкретика.** Есть цифры, детали, имена? (Не "много", а "147")
- [ ] **Эмоция.** Вызывает любопытство, удивление, узнавание?
- [ ] **Незавершённость.** Хочется узнать, что дальше?
- [ ] **Соответствие аудитории.** На каком уровне осведомлённости твой читатель?
- [ ] **Честность.** Пост выполняет обещание хука?
- [ ] **Уникальность.** Это не шаблон, который видели 100 раз?

### Красные флаги:

- ❌ Начинается с "Всем привет" или "Сегодня хочу рассказать"
- ❌ Слишком абстрактно ("Технологии меняют мир")
- ❌ Кликбейт без payoff (обещает больше, чем даёт)
- ❌ Слишком длинное первое предложение (больше 10 слов)
- ❌ Нет причины читать дальше
- ❌ Хук не содержит ни цифры, ни конкретной детали
- ❌ Авторитет заявлен ("я эксперт"), а не встроен ("в нашем исследовании на 776 людях")

## Упражнения для прокачки

### 1. Swipe File

Создай папку/заметку, куда сохраняешь хуки, которые зацепили тебя.

Для каждого записывай:

- Сам хук
- Какой тип (из 6)
- Почему сработало на тебе

### 2. Переписывание

Возьми свой старый пост. Напиши 10 разных хуков к нему:

- 2 истории
- 2 секрета
- 2 проблемы
- 2 обещания
- 2 провокации

### 3. Анализ конкурентов

Найди 5 популярных постов в своей нише. Разбери:

- Какой тип хука использован?
- Какой уровень осведомлённости аудитории предполагается?
- Как бы ты улучшил?

### 4. A/B тест

Один и тот же пост — два разных хука. Публикуй в разное время, сравнивай охваты.

## Шаблоны хуков для фаундер-контента

### Для кейсов:

```
"[Я/Клиент] [результат] [метрика]. Вот что сделали."
```

→ "Вышел с 0 на $2000 MRR за 3 месяца. Вот что сделал."

### Для тактик:

```
"[Боль] — это [X часов/рублей]. [Решение] стоит [Y]."
```

→ "Поиск клиентов через рекламу — это $500 в месяц. Холодные DM стоят $0 и работают лучше."

### Для инсайтов:

```
"[Количество] [действие]. [Вот что понял]."
```

→ "Отправил 100 холодных сообщений за месяц. Вот главный инсайт."

### Для мнений:

```
"[Популярное мнение] — [твоя позиция]. [Почему]."
```

→ "Все говорят: найди инвестора. Я считаю, это ловушка для начинающих. Вот почему."

### Для анонсов:

```
"[Период времени] [что делал]. [Что получилось]."
```

→ "6 месяцев бутстрэпа. 3 проекта. $2000/мес. Без единого рубля инвестиций."
