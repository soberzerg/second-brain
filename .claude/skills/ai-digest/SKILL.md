# AI Daily Digest Skill

## When to use this skill

**Автоматическая активация при:**
- Пользователь просит дайджест, сводку или новости AI за сегодня
- Упоминания: "дайджест", "digest", "сводка", "AI новости", "что нового в AI", "новости за сегодня", "ежедневная сводка"

**Паттерны активации:**
- "Дай дайджест"
- "Что нового в AI?"
- "Сводка за сегодня"
- "AI digest"
- "Покажи новости"

## Instructions

### Источник данных

Файл дайджеста генерируется автоматически скриптом `.scripts/ai-daily-digest.sh` (cron, 9:00 MSK).
Расположение: `500_Research/510_AI_Daily_Digest/YYYY-MM-DD - AI Daily Digest.md`

### Step 1: Проверить наличие файла

Прочитать файл за сегодня:
```
500_Research/510_AI_Daily_Digest/{today} - AI Daily Digest.md
```

**Если файла нет** — сообщить пользователю:

> Дайджест за сегодня ещё не сформирован. Скрипт запускается автоматически в 9:00 MSK.
> Можешь запустить вручную: `.scripts/ai-daily-digest.sh`

**НЕ запускать скрипт самостоятельно.** Только сообщить.

### Step 2: Прочитать и сократить

Прочитать файл целиком, затем выдать **сокращённую сводку** в таком формате:

```
## AI Digest — {дата}

### Главное
- [1-2 предложения на каждую из топ-историй, максимум 5]

### X/Twitter
- [ключевые мнения экспертов — 2-3 пункта, @handle + суть]

### Reddit
- [главные обсуждения — 2-3 пункта, r/subreddit + суть]

### Тренды
- [буллет-лист трендов, как есть в оригинале]

### Для проектов
- [краткие рекомендации по AGIents и блогу, 1-2 пункта каждый]
```

### Принципы сокращения

- **Максимальная плотность**: убрать воду, оставить факты и ссылки
- **Сохранять ссылки**: URL на посты и обсуждения — ключевая ценность
- **Сохранять @handles и r/subreddits**: для быстрой навигации
- **Технические термины на английском**: как в оригинале
- **Русский язык**: вся сводка на русском

### Дополнительно

Если пользователь просит дайджест за другую дату — искать соответствующий файл.
Если просит подробнее по конкретной теме — показать полный раздел из оригинала.
