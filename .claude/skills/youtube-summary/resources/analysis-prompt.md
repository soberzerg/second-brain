# Analysis Prompt for YouTube Videos

Этот промпт используется для анализа YouTube видео через `mcp__gemini-vision__analyze_video`.

## Базовый промпт

```
Проанализируй это видео и создай детальную выжимку на русском языке.

Извлеки следующую информацию:

1. НАЗВАНИЕ ВИДЕО - точное название
2. ОСНОВНАЯ ТЕМА - о чём это видео (1-2 предложения)
3. КЛЮЧЕВЫЕ ТЕЗИСЫ - главные мысли и утверждения автора (5-10 пунктов, каждый с кратким пояснением)
4. ПОЛЕЗНЫЕ ИНСАЙТЫ - неочевидные выводы или интересные наблюдения
5. ПРАКТИЧЕСКИЕ РЕКОМЕНДАЦИИ - что можно применить на практике (если есть)
6. УПОМЯНУТЫЕ РЕСУРСЫ - книги, инструменты, ссылки, имена экспертов (если упоминались)

Важные требования:
- Пиши конкретно, избегай общих фраз типа "автор рассказывает о важности..."
- Сохраняй терминологию автора
- Английские технические термины оставляй как есть
- Для каждого тезиса указывай контекст если он важен
- Если видео техническое - сохраняй точность формулировок
```

## Структура ожидаемого ответа

Gemini должен вернуть структурированный ответ в таком формате:

```
НАЗВАНИЕ ВИДЕО:
[точное название]

ОСНОВНАЯ ТЕМА:
[1-2 предложения]

КЛЮЧЕВЫЕ ТЕЗИСЫ:

1. [Заголовок тезиса 1]
[Пояснение 1-3 предложения]

2. [Заголовок тезиса 2]
[Пояснение 1-3 предложения]

[... и так далее ...]

ПОЛЕЗНЫЕ ИНСАЙТЫ:

• [Инсайт 1]
• [Инсайт 2]
• [Инсайт 3]

ПРАКТИЧЕСКИЕ РЕКОМЕНДАЦИИ:

• [Рекомендация 1]
• [Рекомендация 2]
• [Рекомендация 3]

УПОМЯНУТЫЕ РЕСУРСЫ:

• [Ресурс 1] - [описание]
• [Ресурс 2] - [описание]
```

## Обработка ответа

После получения ответа от Gemini:

1. **Парсинг структуры** - разделить ответ на секции
2. **Валидация** - проверить что все секции заполнены
3. **Форматирование** - преобразовать в markdown по шаблону
4. **Добавление метаданных** - frontmatter с датой и тегами
5. **Очистка** - убрать дубликаты, проверить орфографию

## Fallback для некачественного ответа

Если Gemini вернул неструктурированный ответ:

```typescript
// Попытка 2: упрощённый промпт

Что говорит автор? Какие главные мысли? Что упоминается?

Пиши конкретно, сохраняй технические термины на английском.
`
```
