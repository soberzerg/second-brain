# YouTube Summary Skill

Автоматически создаёт структурированные выжимки из YouTube видео на русском языке.

## Возможности

- 🎬 **Автоактивация** - просто отправь YouTube URL, skill активируется автоматически
- 🇷🇺 **Русский язык** - все пояснения на русском, технические термины - как есть
- 📝 **Структурированный формат** - единый шаблон для всех выжимок
- 💾 **Автосохранение** - файлы в `500_Research/530_YouTube/`
- 🏷️ **Автотеги** - автоматическое проставление тегов по теме
- 🔗 **Obsidian integration** - готовые markdown заметки в vault

## Как использовать

### Автоматическая активация

Просто отправь YouTube URL в чат:

```
https://www.youtube.com/watch?v=dQw4w9WgXcQ
```

Или:

```
https://youtu.be/abc123
```

Skill автоматически:
1. Распознает YouTube URL
2. Проанализирует видео через Gemini Vision
3. Создаст структурированную выжимку
4. Сохранит в `500_Research/530_YouTube/`

### Явный запрос

Можно также явно попросить:

```
Сделай выжимку этого видео: https://youtube.com/watch?v=...
```

```
Проанализируй видео: https://youtu.be/...
```

```
Что в этом видео? [URL]
```

## Что извлекается

Из каждого видео извлекается:

1. **Основная тема** - краткое описание в 1-2 предложениях
2. **Ключевые тезисы** - главные мысли автора (5-10 пунктов с пояснениями)
3. **Полезные инсайты** - неочевидные выводы и наблюдения
4. **Практические рекомендации** - actionable советы (если есть)
5. **Упомянутые ресурсы** - книги, инструменты, эксперты, ссылки

## Ограничения

### Длина видео
- ✅ Короткие видео (<30 мин) - полная обработка
- ⚠️ Средние видео (30-60 мин) - может занять больше времени
- ⚠️ Длинные видео (>1 час) - частичный анализ

### Доступность
- ✅ Публичные видео - работает
- ❌ Приватные видео - не работает
- ❌ Unlisted (по ссылке) - может не работать
- ❌ Региональные ограничения - зависит от доступа

### Качество
- ✅ Чёткая речь - отличный результат
- ⚠️ Музыка + речь - хороший результат
- ❌ Только музыка - пропустится
- ❌ Нет звука - только визуальное описание

## Технические детали

### MCP Integration

Использует `mcp__gemini-vision__analyze_video` для анализа:

```typescript
mcp__gemini-vision__analyze_video({
  youtube_url: "https://youtube.com/watch?v=...",
  prompt: "[detailed analysis prompt]"
})
```

### Обработка времени

- Короткие видео: 30-60 секунд
- Средние видео: 1-2 минуты
- Длинные видео: 2-5 минут

### Fallback стратегия

Если основной анализ не сработал:
1. Попытка с упрощённым промптом
2. Создание файла с пометкой "требует_ревью"
3. Сохранение сырого ответа для ручной обработки

## FAQ

**Q: Можно ли обрабатывать плейлисты?**
A: Пока нет, только отдельные видео. В планах на v1.2.

**Q: Работает ли с видео не на русском?**
A: Да! Выжимка будет на русском, технические термины останутся на языке оригинала.

**Q: Что если видео очень длинное (2+ часа)?**
A: Gemini может обработать только часть. Рекомендуется разбить на несколько коротких видео или использовать временные метки.

**Q: Можно ли изменить формат выжимки?**
A: Да, отредактируй `resources/summary-template.md` под свои нужды.

**Q: Как удалить старые выжимки?**
A: Просто удали файлы из `500_Research/530_YouTube/`. Skill не хранит свою базу данных.

**Q: Можно ли обрабатывать видео с других платформ (Vimeo, TikTok)?**
A: Пока только YouTube. Другие платформы требуют других MCP инструментов.

## Поддержка

Если skill работает некорректно:

1. Проверь что YouTube URL корректный и видео публичное
2. Убедись что MCP server для Gemini Vision настроен
3. Проверь что папка `500_Research/530_YouTube/` доступна для записи
4. Попробуй с другим видео (возможно проблема с конкретным видео)

Если проблема сохраняется - skill создаст файл с пометкой "требует_ревью" и сырым ответом для отладки.
