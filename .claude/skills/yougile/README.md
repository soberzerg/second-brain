# YouGile Task Manager Skill

Навык для автоматического управления задачами в YouGile через MCP-сервер.

## Что делает

Автоматически активируется при упоминании задач, проектов (AGIents, Clario, AGIency, Dogovor24, Amaks) и предоставляет быстрый доступ к:

- Просмотру задач по проектам и доскам
- Созданию новых задач
- Обновлению и завершению задач
- Перемещению задач между колонками

## Автоматическая активация

Навык активируется при упоминании:

- **Задач:** "задачи", "task", "todo", "что в работе", "покажи задачи"
- **Проектов:** AGIents, Amaks, AGIency, Dogovor24, AICodeReview
- **Досок:** Продукт, Sales, CRM, код-ревью
- **Действий:** "создать задачу", "завершить", "переместить", "добавить в бэклог"

## Примеры использования

```
Что у меня в работе?
→ Показывает задачи AGIents - Продукт - В работе

Покажи задачи Amaks
→ Показывает задачи клиента Amaks

Добавь задачу "Исправить баг в API" в бэклог
→ Создаёт задачу в AGIents - Продукт - Бэклог

Что в CRM?
→ Показывает лиды в AGIency - Sales - First touch

Завершить задачу #task-123
→ Помечает задачу как завершённую
```

## Умные дефолты

Навык использует контекст из CLAUDE.md для определения приоритетов:

- **Проект по умолчанию:** AGIents (приоритетный проект)
- **Доска по умолчанию:** основная доска проекта
- **Колонка по умолчанию:** "В работе"

## Quick Aliases

Навык распознаёт контекст из естественных фраз:

| Фраза | Проект | Доска | Колонка |
|-------|--------|-------|---------|
| "задачи продукта" | AGIents | Продукт | В работе |
| "amaks", "санатории" | AGIents | Amaks | В работе |
| "crm", "лиды", "продажи" | AGIency | Sales | First touch |
| "dogovor", "консалтинг" | Заказная | Dogovor24 | В работе |
| "код-ревью" | Заказная | AICodeReview | В работе |

## Интеграция с другими навыками

- **daily-review**: Задачи включаются в ежедневный обзор
- **thinking-partner**: Обсуждение приоритизации задач
- **weekly-synthesis**: Статистика задач в недельном обзоре

## Обновление структуры проектов

Если добавляются новые проекты, доски или колонки в YouGile:

1. Получить ID через MCP: `list_projects`, `list_boards`, `list_columns`
2. Обновить [project-structure.md](resources/project-structure.md)
3. При необходимости добавить новые Quick Aliases в SKILL.md
