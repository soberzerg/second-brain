# YouTube Video Summary

Создание структурированной выжимки из YouTube видео на русском языке.

## Входные данные

**URL видео:** $ARGUMENTS

## Задача

### Шаг 1: Валидация URL

Проверь что URL содержит `youtube.com` или `youtu.be`. Если URL некорректный или не передан, попроси пользователя предоставить корректную ссылку.

### Шаг 2: Анализ видео

Вызови `mcp__gemini-vision__analyze_video` с параметрами:

```
youtube_url: [URL из $ARGUMENTS]
prompt: |
  Проанализируй это видео и создай детальную выжимку на русском языке.

  Извлеки следующую информацию:

  1. НАЗВАНИЕ ВИДЕО - точное название
  2. ОСНОВНАЯ ТЕМА - о чём это видео (1-2 предложения)
  3. КЛЮЧЕВЫЕ ТЕЗИСЫ - главные мысли и утверждения автора (5-10 пунктов, каждый с кратким пояснением)
  4. ПОЛЕЗНЫЕ ИНСАЙТЫ - неочевидные выводы или интересные наблюдения
  5. ПРАКТИЧЕСКИЕ РЕКОМЕНДАЦИИ - что можно применить на практике (если есть)
  6. УПОМЯНУТЫЕ РЕСУРСЫ - книги, инструменты, ссылки, имена экспертов (если упоминались)

  Важные требования:
  - Пиши конкретно, избегай общих фраз типа "автор рассказывает о важности..."
  - Сохраняй терминологию автора
  - Английские технические термины оставляй как есть
  - Для каждого тезиса указывай контекст если он важен
  - Если видео техническое - сохраняй точность формулировок
```

### Шаг 3: Форматирование результата

Оформи результат в следующем формате (это будет содержимое файла):

```markdown
---
tags:
  - youtube
  - выжимка
source: "[URL видео]"
date: [текущая дата в формате YYYY-MM-DD]
---

# Выжимка: [Название видео]

**Источник:** [URL видео]

## Основная тема

[1-2 предложения о чём видео]

## Ключевые тезисы

1. **[Тезис 1]** - [краткое пояснение]
2. **[Тезис 2]** - [краткое пояснение]
3. ...

## Полезные инсайты

- [Инсайт 1]
- [Инсайт 2]
- ...

## Практические рекомендации

- [ ] [Рекомендация 1]
- [ ] [Рекомендация 2]
- ...

## Упомянутые ресурсы

- **[Название ресурса]** - [краткое описание]
- ...

---
*Создано с помощью /youtube-summary*
```

### Шаг 4: Сохранение в Obsidian

1. Проверь существует ли папка `03_Resources/YouTube-выжимки/` (используй Glob)
2. Если папки нет - она будет создана автоматически при записи файла
3. Сохрани файл с именем: `03_Resources/YouTube-выжимки/YYYY-MM-DD - [Название видео].md`
   - Название видео очисти от спецсимволов: `/ \ : * ? " < > |`
   - Если название слишком длинное (>50 символов), обрежь и добавь `...`

### Шаг 5: Подтверждение

Сообщи пользователю:
- Что выжимка создана
- Путь к сохранённому файлу
- Краткое резюме основной темы видео

## Примеры использования

```
/youtube-summary https://www.youtube.com/watch?v=dQw4w9WgXcQ
/youtube-summary https://youtu.be/dQw4w9WgXcQ
```

## Важные замечания

- Обработка видео может занять 30-60 секунд
- Длинные видео (>1 час) могут быть обработаны частично
- Качество выжимки зависит от чёткости речи в видео
- Работает только с публичными YouTube видео
- Если видео недоступно или приватное - сообщи об этом пользователю
