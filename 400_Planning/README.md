# Planning

Планирование и рефлексия: roadmap, недельные планы, синтезы и ежедневные заметки.

## Структура

- `410_Weekly_Plans/` — недельные планы с целями и приоритетами
- `420_Weekly_Syntheses/` — итоги недели, выводы, корректировка курса
- `430_Daily_Notes/` — ежедневные заметки и рефлексия
- `440_Roadmap/` — личный роадмэп и квартальные/годовые планы

## Ритм планирования

| Цикл | Что делаем | Где |
| ---- | ---------- | --- |
| Ежедневно | Заметки, прогресс, мысли | `430_Daily_Notes/` |
| Еженедельно | План на неделю | `410_Weekly_Plans/` |
| Еженедельно (конец) | Синтез итогов недели | `420_Weekly_Syntheses/` |
| Ежеквартально / ежегодно | Стратегические цели | `440_Roadmap/` |
