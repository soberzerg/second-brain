## Major Insights

### 1. Confidence Meter как Decision-Making Tool

**Контекст:** GIST Framework Confidence Meter использован для AGIents.pro

**Insight:**
Переход от gut feeling к evidence-based decisions трансформирует quality принятия решений:
- **До конкурентного анализа:** 3/10 - "слишком рано для кода"
- **После анализа:** 7/10 - "готовы к MVP"
- **Измеримые шаги:** каждое действие увеличивает confidence на основе evidence

**Применение:**
- Не начинать coding пока Confidence < 7/10
- Документировать, какие actions увеличат confidence
- Использовать для всех major decisions (не только продуктовых)

### 2. Templates-First Approach Validated

**Контекст:** Все успешные платформы (Make, Landbot, ManyChat) используют templates

**Insight:**
Users prefer **quick start over blank canvas**:
- Templates = faster time-to-value (< 8h goal achievable)
- Visual builder можно добавить позже
- Templates → use cases → more templates (flywheel)

**Противоречие resolved:** "No-code builder vs Templates-first" → Start with templates, add builder v2.0

**Validation:** ICE score 7.8 для templates был правильным решением.

### 3. AGIents = Platform Category, Not Just Automation Tool

**Контекст:** Конкурентный анализ показал 3 отдельные категории (workflow, conversational, AI agents)

**Insight:**
AGIents.pro занимает **уникальную позицию** как **Hybrid Platform**:
- Workflow automation (как Make/Zapier)
- Conversational AI (как Voiceflow/Landbot)
- AI Agents (как Stack AI/Relevance AI)
- All-in-one вместо 3 разных инструментов

**Positioning:** "Единая платформа для создания любых ИИ-агентов" (не "еще один workflow tool")

### 4. Self-Hosted = Strategic Differentiator

**Контекст:** 2/16 платформ предлагают self-hosted (Stack AI, Relevance AI - дорогие enterprise)

**Insight:**
Self-hosted для Russian market = **must-have**, не nice-to-have:
- Data sovereignty requirements
- Sanctions compliance
- Trust & privacy concerns
- Pricing advantage (self-hosted = lower costs)

**Opportunity:** Единственная **доступная** self-hosted платформа для Russian SMB.

### 5. Flywheel Effect: AGIency + AGIents

**Контекст:** Два проекта взаимно усиливают друг друга

**Insight:**
AGIency использует AGIents → создает templates → привлекает users → генерирует кейсы → новые templates:

```
AGIency клиенты (Amaks)
    ↓
Real use cases → Templates
    ↓
AGIents users → Больше кейсов
    ↓
Лучшие templates → Проще onboarding
    ↓
Больше AGIency клиентов
```

**Strategic implication:** Не разделять проекты - это **единая экосистема**.
