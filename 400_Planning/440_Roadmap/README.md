# Roadmap

## AI-Native Plan

Цель: создать AI-native компанию

### Критерии

- Все процессы, которые могут быть переведены на AI, переведены на AI
- Все сотрудники работают с AI: умеют создавать mcp, навыки, агентов, правильно работать с контекстом
- В компании внедрён "второй мозг" с разграничением прав, все сотрудники с ним работают
- Есть общая библиотека промтов для выполнения редко повторяющихся задач
- AI используется тремя способами:
  1. Вручную сотрудниками - deep research, агенты, также и простые "чатботы"
  2. AI встроен в различные флоу автоматизации
     - по расписанию (ежедневные синки, дневные и недельные отчёты)
     - по триггерам (внутренние чаты, чаты клиентов)
     - в виде внешних чатботов: продажи, поддержка
     - в виде голосовых ботов для продаж
  3. В виде автономных агентов, которые встроены в рабочую среду: берут и выполняют задачи, проактивно

### KPI разработчиков

- 10 PR в неделю для парттаймеров
- 25 PR в неделю для фуллтаймеров

#### Как этого добиться?

1. Узкие горлышки:
   - постановки задач
   - тестирование
   - code review
2. Решения:
   - второй мозг у PM, генерация небольших гипотез
   - генерация небольших гипотез самими разработчиками
   - авто-тесты на всё: unit-тесты, интеграционные, e2e
   - pre-push code-review с помощью 2 разных агентов
   - автоматизированное PR review с помощью 2 агентов
   - 20% времени на рефакторинг (для исключения деградации кодовой базы)

### KPI продакта

- 1 мини-продукт в неделю
