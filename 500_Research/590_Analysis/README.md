# Analysis

Внутренне-производный анализ, синтез и сравнения — результаты запросов и исследований, выполненных в рамках сессий Claude Code.

## Назначение

В отличие от `580_Web_Research/` (внешние источники), здесь хранятся **внутренние** аналитические артефакты:

- Сравнительный анализ (технологий, подходов, конкурентов)
- Синтез данных из нескольких источников vault
- Кросс-проектные исследования
- Ответы на сложные запросы, достойные сохранения

## Как попадают файлы сюда

1. Claude создаёт ценный анализ в рамках сессии
2. Предлагает сохранить результат в wiki
3. Файл создаётся с `type: analysis` в frontmatter
4. Обновляются `index.md` и `log.md`

## Именование файлов

`YYYY-MM-DD - Тема анализа.md`
