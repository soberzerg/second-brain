# Транскрипция: Коучинг-сессия - стратегия продаж ИИ-бот и развитие бизнеса

**Источник:** video1074641009.mp4
**Длительность:** 01:31:20
**Дата транскрипции:** 2026-02-06

## Основная тема

Коучинг-сессия Сергея (Серега) с наставником Маратом. Обсуждение стратегии развития ИИ-бот бизнеса: текущие клиенты, ценообразование, делегирование разработки vs. продаж, неопределенность с Кларио (инвестиции), выход на рынок санаториев/отелей, важность нетворкинга и контактной базы, роль соцсетей для охватов.

## Ключевые тезисы

1. **Цель 300 тысяч на своем продукте** - Сергей ставит цель заработать 300К рублей на ИИ-бот. Для этого нужен либо один крупный проект, либо 3-4 средних. Текущие клиенты дают ~80-100К.

2. **Проблема "разработчик vs. продажи"** - Сергей сам разрабатывает платформу и не может одновременно активно продавать. Нанимать разработчика за 300К/мес нецелесообразно на текущем этапе. Решение: обучить ассистента внедрению, а самому фокусироваться на продажах.

3. **Неопределенность с Кларио** - Ожидание инвестиций (KYC процедура от акселератора). Результат должен быть на следующей неделе. Сергей не должен ставить свои планы в зависимость от решения по Кларио.

4. **Вайб-кодинг как ускоритель** - Сергей использует AI для разработки платформы, что значительно быстрее, чем объяснять задачи другому разработчику. Но джуны могут "навайб-кодить параши", которую потом придётся разгребать.

5. **Стратегия выхода на санатории** - Уже есть успешный кейс (проект за 3.6 млн). Нужно: собрать контакты, подготовить email-рассылку, затем обзвон. Лендинг желателен, но не обязателен.

6. **Позиция предпринимателя** - Марат подчеркивает: нужно занять позицию "мне тотально туда надо", убрать все барьеры. Не должно быть оправданий вроде "не могу продавать, потому что не вывезу объем".

7. **Контактная база - фундамент продаж** - Базовый канал продаж - личная контактная база. Регулярно писать старым знакомым (BM, Спарта, коучинг), актуализировать связи. 5 людей в день - минимум.

8. **Соцсети для охватов, не для прямых продаж** - Задача соцсетей - максимальные охваты и подписчики. Не нужно таргетировать ЦА, нужно брать хайповые темы из ИИ-ниши. Продажи строятся уже поверх охватов через воронки.

9. **Делегирование внедрения** - Ассистент может обучиться делать промпт-инженеринг, настраивать агентов и базы знаний. Это освобождает Сергея для продаж без необходимости нанимать дорогого разработчика.

10. **Текущая клиентская воронка** - 3 активных заявки + санаторий в работе. Источники: нетворкинг на курсах, бывший партнер, случайное знакомство. Рынок отвечает на действия Сергея.

## Полезные инсайты

- Продаешь не продукт (ИИ-боты), а смыслы и решения проблем клиента (продажи, автоматизация, экономия). Никому не нужны "ИИ-боты" - людям нужны деньги и автоматизация.
- Аналогия Марата: "вода протекает в любую щелочку" - если намерение сильное, возможности найдутся.
- Даже "корявая" рассылка с днем рождения лучше, чем полное отсутствие контакта - это создает предысторию для будущей коммуникации.
- Транскрипция встреч (продукт типа Fireflies) - потенциальный B2C продукт с большим спросом, который может стать входной точкой для B2B продукта Кларио.
- 90-дневный марафон Reels: первые 2 месяца - отписки, на 3-й месяц - рост (+300 подписчиков). Ключевая ошибка - остановился после марафона.

## Практические рекомендации

- [ ] Обучить ассистента внедрению ботов и базы знаний (промпт-инженеринг, настройка агентов)
- [ ] Собрать контактную базу санаториев/отелей (email, телефоны)
- [ ] Подготовить email-шаблон для холодной рассылки по санаториям
- [ ] Сделать лендинг для ИИ-бот услуг (через ассистента: конкурентный анализ -> тексты -> верстка)
- [ ] Пробежаться по старой контактной базе: выписать людей, актуализировать связи (5 человек/день)
- [ ] Завтра: созвон с интернет-магазином моющих средств, закрыть на 300-400К
- [ ] Сделать конспект этой сессии и провалидировать
- [ ] Задать привычку: каждый день писать 5 старым контактам
- [ ] Ассистенту: подготовить демо-видео платформы, конкурентный анализ по санаториям

## Упомянутые ресурсы

- **Savvy** - платформа-конкурент для создания чат-ботов, можно использовать как партнер
- **Fireflies** - сервис транскрибации встреч, потенциальный B2C продукт для Кларио
- **TravelLine** - система бронирования для санаториев/отелей, с которой есть интеграция
- **BM (Бизнес-молодость)** - нетворкинг-база контактов Сергея
- **Спарта** - технические тренинги, источник контактов
