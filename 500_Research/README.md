# Research

Исследования, ресурсы и база знаний.

## Структура

| Папка | Содержание |
| ----- | ---------- |
| `510_AI_Daily_Digest/` | Ежедневные дайджесты AI-новостей |
| `520_Social_Research/` | Исследования из X/Twitter, Reddit |
| `530_YouTube/` | Заметки и выжимки из видео |
| `540_Transcriptions/` | Транскрипции встреч и записей |
| `550_AI/` | ИИ-технологии: агенты, RAG, workflows |
| `560_Business_Models/` | Бизнес-фреймворки и стратегии |
| `570_CustDev/` | Вопросы и методики CustDev (B2B/B2C) |

## Принципы организации

- **По теме, не по источнику** — группируем по смыслу
- **Evergreen > Ephemeral** — долгоживущие заметки важнее новостей
- **Атомарные заметки** — одна концепция = одна заметка
- **Связи** — линковать между собой и с проектами

## Как использовать

1. AI-дайджесты (`510`) — ежедневный мониторинг трендов
2. Social Research (`520`) — анализ обсуждений и мнений
3. YouTube (`530`) — выжимки из образовательных видео
4. Бизнес-модели (`560`) — фреймворки для проектов из [[200_Projects/README|200_Projects]]
5. CustDev (`570`) — вопросы для интервью с клиентами
