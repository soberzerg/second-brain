# Выжимка: ClaudeClaw The Sequel - Build Your AI Assistant With Claude Code

**Источник:** https://www.youtube.com/watch?v=rVzGu5OYYS0
**Дата:** 2026-04-16

## Основная тема

ClaudeClaw — персонализированная операционная система для управления AI-агентами на базе Claude Code. Автор показывает, как построить собственный кастомизируемый центр управления, интегрированный с различными интерфейсами (телефон, браузер, десктоп) и сервисами, используя Anthropic Agent SDK и существующую подписку Claude Code.

## Ключевые тезисы

1. **ClaudeClaw — персонализированный Command Center** - Это не готовое решение вроде OpenClaw или Hermez, а настраиваемый центр управления, построенный на основе подписки Claude Code. Оркестрирует множество специализированных AI-агентов через единый интерфейс.

2. **"War Room" — голосовой совет из 5 агентов** - Функция голосового общения с советом из пяти специализированных AI-агентов (Main, Research, Comms, Content, Ops). Главный агент (Main) координирует, распределяя запросы по специалистам. Агенты делегируют задачи друг другу.

3. **Agent SDK как фундамент архитектуры** - Вся система построена на Anthropic Agent SDK — бесплатном фреймворке, который позволяет запускать удалённые терминальные сессии Claude Code (CLI) с любого устройства. SDK — "мост" между пользовательскими устройствами и Claude Code.

4. **Эволюция архитектуры от V0** - V0 была минимальной: телефон → Telegram → Mac Mini + Node.js + 200 строк кода. Текущая версия значительно расширена:
   - **Интерфейсы:** телефон, браузер, дашборд
   - **Каналы:** Telegram, WhatsApp, Slack, Discord
   - **Ядро:** Message Queue, Classifier, 5-layer Memory retrieval
   - **Voice I/O:** Greg Whisper, ElevenLabs, Kokoro, macOS say
   - **War Room:** Gemini Live, Deepgram, Cartesia, PipeCat, Silero VAD
   - **Meeting Bot:** Pika video avatar, Daily.co voice, pre-flight briefing (Gmail + Calendar + Memory)

5. **Hive Mind — коллективный разум агентов** - Унифицированная память, которая отслеживает каждую задачу всех агентов. Позволяет обмениваться информацией и поддерживать единый контекст. Интегрируется со Scheduler для управления задачами.

6. **Многоуровневая система памяти** - Gemini 3 Flash динамически анализирует разговоры и классифицирует информацию на "факты", "предпочтения" и "контекст". Данные хранятся в SQLite Memory + Obsidian notes. Есть "закреплённые" (pinned) воспоминания (персистентные) и временные (затухающие).

7. **Автоматизация задач через дашборд** - Задачи создаются через интерфейс и автоматически назначаются наиболее подходящему агенту с помощью LLM (Gemini). Переводятся из "queued" в "live" для выполнения.

8. **Многослойная безопасность** - Chat ID Allowlist, PIN Lock, Idle Auto-Lock, Kill Phrase, Exfil Guard (сканер секретов), Audit Log — критически важно для защиты данных при работе с персонализированными AI-ассистентами.

9. **Устойчивость к изменениям политики Anthropic** - 4 апреля Anthropic запретил коммерческое использование подписки через сторонние оболочки (OpenClaw). Но Agent SDK для личных локальных инструментов остаётся разрешённым. Собственное решение наследует улучшения Anthropic, не привязываясь к сторонним продуктам.

## Полезные инсайты

- **Персонализация памяти — ключ к эффективности** - В отличие от универсальных систем, ClaudeClaw демонстрирует, что система памяти должна быть глубоко индивидуализирована под конкретные рабочие процессы и предпочтения пользователя.

- **Модульность + мобильность через Cloudflare Tunnel** - Mac Mini как хост + Cloudflare Tunnel для удалённого доступа обеспечивают работу с AI-системой через телефон, браузер или Telegram из любой точки.

- **Agent SDK > сторонние платформы** - SDK даёт максимальный контроль над функциональностью и стоимостью, защищая от нестабильности или изменений условий сторонних платформ.

- **PipeCat как оркестратор голосового пайплайна** - Open-source фреймворк (Python) для оркестрации STT → LLM → TTS микросервисов. Позволяет гибко комбинировать Deepgram (STT) + Cartesia (TTS) + LLM.

- **Классификатор + очередь предотвращают "тихие сбои"** - Message Queue и Classifier обеспечивают упорядоченную обработку даже при одновременной работе нескольких агентов или задач.
