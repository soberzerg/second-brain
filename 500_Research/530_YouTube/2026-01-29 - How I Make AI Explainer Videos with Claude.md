# Выжимка: How I Make AI Explainer Videos with Claude

**Источник:** https://youtu.be/pSnTKR2Phyo?si=wl4PZ0ZWDywjAVyX

## Основная тема

Видео демонстрирует, как использовать новый skill Remotion для Claude, чтобы создавать и редактировать различные типы видеоконтента (продуктовые демо, моушн-графику, YouTube-видео и презентации) с помощью текстовых промптов. Само видео полностью создано таким способом.

## Ключевые тезисы

1. **Remotion как skill для Claude** - Claude получил возможность генерировать видео напрямую из текстовых промптов через фреймворк Remotion

2. **Prompt-driven workflow** - весь процесс создания и редактирования видео управляется текстовыми командами, без традиционного ручного редактирования

3. **Три уровня работы:**
   - Level 1: Setup - установка Claude Desktop и подключение skills через `npx skills add remotion-dev/skills`
   - Level 2: First Videos - создание видео с нуля, используя свой футаж и графику
   - Level 3: Skill stacking - добавление voiceovers, генерация визуалов, поддержание стиля

4. **Аналогия с шеф-поваром** - Claude без навыков может только описать видео, с Remotion skill он получает "полностью укомплектованную кухню" для создания

5. **CLAUDE.md для кастомизации** - файл для хранения брендбука, стилей, API keys и предпочтительных инструментов. Claude использует его как "блокнот" для контекста

6. **Стэкинг навыков** - интеграция с ElevenLabs для voiceovers и Nano Banana Pro для генерации изображений позволяет создавать насыщенный брендированный контент

7. **Модульное создание длинных видео** - для YouTube используются независимые компоненты (intro, main content, outro), которые Claude "сшивает" вместе с переходами

## Полезные инсайты

- Само видео - живая демонстрация подхода, полностью смонтировано через AI-промптинг
- Порог входа для нетехнических специалистов значительно снижается - достаточно "разговаривать с Claude"
- CLAUDE.md позволяет AI непрерывно обучаться и адаптироваться к предпочтениям пользователя
- Claude может подтягивать цвета, шрифты и изображения из кодовой базы сайта для соответствия брендбуку

## Практические рекомендации

- [ ] Установить Claude Desktop и переключиться на вкладку "Code"
- [ ] Добавить Remotion skill командой `npx skills add remotion-dev/skills`
- [ ] Создать папку для проектов (например, `remotion-videos`)
- [ ] Создать файл CLAUDE.md с брендбуком, API keys (ElevenLabs, Nano Banana Pro) и инструкциями
- [ ] Для продуктового демо: передать URL сайта + описание желаемого видео
- [ ] Для своих assets: перетащить файл в промпт
- [ ] Для voiceovers: добавить ElevenLabs API key в CLAUDE.md, указать модель V3
- [ ] Для длинных видео: разбить на модули, записать футаж, попросить Claude "сшить"

## Упомянутые ресурсы

- **Claude Desktop** - приложение для локального запуска Claude Code
- **Remotion** - JavaScript-фреймворк для создания видео через React
- **Node.js** - необходим для работы Claude Code
- **ElevenLabs** - платформа для генерации voiceovers (модель V3)
- **Nano Banana Pro** - инструмент для генерации изображений
- **WaveSpeedAI** - для получения API ключа ElevenLabs
- **bookedin.ai** - сайт, использованный как пример для продуктового демо
- **CLAUDE.md / AGENT.md** - файл для хранения кастомных инструкций
