# Clario — ИИ-аналитика продаж

B2B платформа для аналитики продаж с использованием ИИ.

## Моя роль

**CTO-кофаундер:**

- Собрал и руковожу технической командой
- Разработка архитектуры (Go + Python, микросервисы)
- DevOps и процессы разработки
- Стратегическое техническое развитие

## Направления

- Аналитика продаж
- ИИ-инсайты для B2B команд
- Платформенное решение

## Структура папок

- `231_Roadmaps/` — дорожные карты
- `232_Architecture/` — архитектура системы
- `233_PRDs/` — спецификации продукта

- Отдельная компания (не связана с AGIents)
- Командная разработка (vs соло в AGIents)
- B2B сегмент
