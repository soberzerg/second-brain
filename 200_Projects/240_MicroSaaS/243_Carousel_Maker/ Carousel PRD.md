# Nano Banana Carousel Maker - Masterplan (SaaS)

## Обзор приложения

**Nano Banana Carousel Maker** — это SaaS-платформа для автоматической генерации вирусных каруселей для социальных сетей (LinkedIn, Instagram). Продукт позволяет пользователям создавать профессиональные слайды за минуты, используя AI для генерации структуры, текстов и изображений. Главная особенность — поддержка фирменного стиля "Nano Banana" (высококонтрастный желто-черный стиль) как флагманского пресета, а также возможность настройки собственных стилей.

## Цели и задачи

1.  **Автоматизация:** Сократить время создания качественной карусели с часов до минут.
2.  **Доступность:** Предоставить доступ к PRO-дизайну и копирайтингу пользователям без навыков дизайна.
3.  **Масштабируемость:** Построить надежную SaaS-архитектуру, способную обрабатывать пиковые нагрузки при генерации изображений.
4.  **Монетизация:** Реализовать гибкую систему оплаты за использование (Pay-as-you-go кредиты).

## Целевая аудитория

*   **Основной сегмент:** SMM-специалисты, Маркетологи, Основатели стартапов (Founders), Личные бренды.
*   **Вторичный сегмент:** Контент-мейкеры, инфобизнесмены.

**User Persona:**
*Alex, фаундер AI-стартапа.* Ему нужно постить в LinkedIn 3 раза в неделю для роста аудитории. У него нет времени на Figma и денег на дизайнера. Он хочет ввести тему поста и получить готовую, красивую карусель, которую не стыдно выложить.

## Основные фичи

### MVP (Фаза 1) — Бот в телеграм

1. **Система Кредитов:** Покупка пакетов кредитов, списание за каждую генерацию карусели.
2. **Режим генерации:**
    *   **Быстрый старт:** Ввод текста -> Готовая карусель.
3.  **AI-Генератор:**
    *   Генерация текстов слайдов (Copywriting).
    *   Генерация изображений (Nano Banana) под выбранный стиль.
4.  **Библиотека стилей:**
    *   Базовые пресеты (Minimalist, Tech, Corporate, Nano Banana).
5.  **Экспорт:** Скачивание готовой карусели изображениями в формате PNG (1080x1350).

### Будущее развитие (Фаза 2) - Веб-приложение

1.  **Аутентификация и Профиль:** Регистрация (Gmail), личный кабинет, история генераций.
2.  **Новый режим генерации:**
    *   **Конструктор (Wizard):** Пошаговый выбор (Тема -> Стиль -> Структура -> Контент).
3.  **AI-Генератор:**
    *   Генерация структуры (Hook, Problem, Solution, CTA).
4.  **Редактор карусели:** Возможность вручную отредактировать текст и заменить картинку на каждом слайде.
5.  **Библиотека стилей:**
    *   Кастомный стиль (выбор цветов/шрифтов).
6.  **Экспорт:** Скачивание готовой карусели архивом (ZIP).
7. Командный доступ (Team seats).
8. Аналитика популярности тем.

## Технический стек

| Компонент | Технология | Обоснование |
| :--- | :--- | :--- |
| **Frontend** | **React** | Стандарт индустрии, богатая экосистема компонентов для интерактивных редакторов. |
| **Backend** | **FastAPI (Python 3.13)** | Высокая производительность (async), нативная работа с AI/ML библиотеками, строгая типизация (Pydantic). |
| **Package Manager** | **UV** | Современный, сверхбыстрый менеджер пакетов для Python. |
| **Database** | **PostgreSQL** | Надежная реляционная БД. Драйверы: `asyncpg` (для асинхронности) + `psycopg2-binary`. |
| **ORM** | **SQLAlchemy 2.x** + **Alembic** | Современный стандарт для работы с БД и миграциями в Python. |
| **Async Tasks** | **Celery 5.4** + **Redis** | Обработка тяжелых задач (генерация картинок) в фоне. Redis как брокер сообщений и кэш. |
| **Containerization** | **Docker** & **Docker Compose** | Изолированная среда, удобство развертывания и локальной разработки. |
| **Storage** | **S3-compatible (MinIO/AWS)** | Хранение сгенерированных изображений и ассетов. |

## UI/UX Принципы

*   **Focus on Result:** Пользователь должен видеть превью результата как можно раньше.
*   **Wizard-based:** Сложный процесс разбит на простые, понятные шаги.
*   **Mobile-friendly:** Возможность просмотреть и скачать карусель с телефона.
*   **Обратная связь:** Четкая индикация процессов генерации (лоадеры, прогресс-бары), так как AI работает не мгновенно.
