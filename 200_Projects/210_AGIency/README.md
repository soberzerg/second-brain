# AGIency.pro — AI-агентство

Агентство, специализирующееся на автоматизации бизнес-процессов с помощью ИИ-агентов.

## Бизнес-модель

- Консалтинговые услуги по внедрению ИИ
- Реализация автоматизаций для клиентов
- Использует платформу [[200_Projects/220_AGIents/README|AGIents.pro]] как основной инструмент

## Услуги

- Создание кастомных ИИ-агентов для клиентов
- Автоматизация бизнес-процессов
- Консалтинг по внедрению ИИ
- Express Security Audit (AI-код-ревью)

## Структура папок

- `211_Консалтинг/` — консалтинговые проекты
- `212_Express_Security_Audit/` — экспресс-аудит безопасности кода
- `213_Second_Brain/` — проект "Второй мозг" для клиентов
- `214_Prospecting/` — поиск и привлечение клиентов
- `215_Case_Studies/` — кейсы реализованных проектов
- `216_Clients/` — материалы по клиентам
- `219_HR/` — найм и команда
