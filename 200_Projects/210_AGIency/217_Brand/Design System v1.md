# AGIency — Дизайн-система v1.0

Бренд-документ AGIency. Формула: **светлая база + инженерная строгость + один сильный акцент**. Дизайн-система отвергает визуальные клише AI-рынка (фиолетовые градиенты, неон, стекломорфизм) в пользу ощущения «не магия ИИ — инженерия результата».

## 1. Основа бренда

- **Позиционирование**: инженерное агентство ИИ-автоматизации. Проектирует и внедряет ИИ-ассистентов, чат-ботов и n8n-сценарии, встраиваемые в реальные бизнес-процессы.
- **Формула бренда**: Понимание → Конструкция → Результат.
- **Архетипы**: Мудрец → Творец → Герой.
  - **Мудрец** — смысл, логика, ясность.
  - **Творец** — визуальный язык, проектирование, форма.
  - **Герой** — собранность, действие, доведение до результата.
- **Слоган**: «Не магия ИИ. Инженерия результата.»

## 2. Принципы дизайн-системы

| Принцип | Суть |
|---|---|
| **Системность** | каждый элемент служит логике интерфейса |
| **Минимализм** | только смысл и структура, ничего лишнего |
| **Инженерность** | графика как схема, а не декор |
| **Фокус на результате** | акценты дозированно, по делу |
| **Читаемость** | интерфейс понятен даже без пояснений |

## 3. Цветовая система

Светлая база и жёсткий акцент. Отказ от фиолетового и AI-клише.

| Роль | HEX | Использование | Примечание |
|---|---|---|---|
| Основной фон | `#FAF8F5` | Фон сайта, большие поля | Тёплый светлый, не чисто белый |
| Светло-серый | `#F1EFEA` | Вторичные блоки, поверхности | Ритм и глубина |
| Серый 300 | `#D9D3CC` | Линии, разделители, сетка | Низкий контраст |
| Текст / графит | `#1F1F1F` | Основной текст, крупные заголовки | База визуальной строгости |
| Вторичный текст | `#5F5F5F` | Описания, вторичные подписи | Сдержанная контрастность |
| **Акцент красный** | `#E53935` | CTA, активные состояния, ключевые узлы | Использовать редко и осознанно |

**Правила использования цвета:**
- Основа интерфейса — светлая и спокойная.
- Красный — только для действия, результата и ключевых точек внимания.
- Чёрный/графит создаёт инженерную строгость.
- Не использовать градиенты, свечения, неон, стекломорфизм.

## 4. Типографика

Ясная, современная, технологичная типографика без декоративности. Основной шрифт — **Manrope**.

| Роль | Шрифт | Применение | Параметры |
|---|---|---|---|
| Основной | Manrope | Интерфейс, тексты, кнопки, цифры | Regular / Medium / Semibold |
| H1 | Manrope | Главные заголовки | Bold, крупный кегль, плотная строка |
| H2 | Manrope | Заголовки секций | Semibold |
| Body | Manrope | Основные абзацы | Regular |
| Caption | Manrope | Подписи, технические метки | Regular, малый кегль |

**Типографические правила:**
- Большие заголовки с сильной иерархией.
- Короткие, уверенные формулировки.
- Много воздуха между смысловыми блоками.
- Текст помогает понимать систему, а не украшает её.

## 5. Логотип

Знак строится на выделении **‘AGI’** внутри названия **AGIency** — подчёркивает связь с ИИ, сохраняя инженерный характер.

- **‘AGI’** — чёрный/графит, более визуально весомый.
- **‘ency’** — облегчённый, нейтральный, поддерживающий.
- **Красная точка** в букве A — единственный акцент.
- Без градиентов, свечения, объёмов и лишних эффектов.
- Логотип корректно работает на светлом фоне.
- Допустимые версии: горизонтальная, монохромная, компактный знак.

**Правила использования логотипа:**
- Предпочтительный фон — светлый.
- Сохранять свободное пространство вокруг логотипа.
- Не растягивать, не деформировать, не накладывать эффекты.
- Красную точку не заменять другими акцентами без системы.

## 6. UI-язык

Интерфейс ощущается как **инженерный, чистый и точный**.

- **Основной мотив** — схемы, оси, сетки, техничные линии.
- **Карточки** — простые, с тонкими границами и большим количеством воздуха.
- **Иллюстрации** — не декоративные, а структурные: блоки, узлы, уровни, диаграммы.
- **Сетка** — строгая, с ощутимой ритмикой.
- **Акцентные элементы** — точки, линии, стрелки, метки процесса.

### Компоненты

| Компонент | Правило |
|---|---|
| Кнопки primary | Красная (`#E53935`) с белым текстом |
| Кнопки secondary | Контурная / нейтральная |
| Карточки | Белый или светло-серый фон, тонкая граница, минимум визуального шума |
| Метрики | Крупные цифры, простая подпись, без лишних иконок |
| Навигация | Спокойная, чистая, без навязчивости |
| Диаграммы | Монохромная база + красный акцент на ключевых узлах |

## 7. Тон визуальной коммуникации

- Спокойный, интеллектуальный, собранный.
- Без «вау-эффектов» ради самих эффектов.
- Визуал должен **объяснять, а не впечатлять**.
- Каждый акцент должен быть оправдан логикой.

## 8. Запреты

- ❌ Фиолетовые градиенты и старые AI-клише.
- ❌ Свечение, неон, стеклянные эффекты, псевдо-3D.
- ❌ Перегруз интерфейса иконками и декоративными паттернами.
- ❌ Красный как массовый декоративный цвет.

## 9. Формула для команды

> Дизайн-система AGIency = **светлая база + инженерная строгость + один сильный акцент**.
>
> **Проверка на соответствие:** если убрать текст, интерфейс всё равно должен объяснять смысл.
