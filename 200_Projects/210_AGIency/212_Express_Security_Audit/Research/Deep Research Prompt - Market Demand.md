Я планирую запустить услугу "Express Security Audit" — AI-powered аудит безопасности кода за $200 с доставкой отчёта за 24-48 часов. Целевая аудитория: стартапы, небольшие dev-команды, фрилансеры с клиентскими проектами. Фокус на PHP/Laravel, но потенциально расширение на другие языки.

## Вопросы для исследования

### 1. Размер рынка и спрос
- Какой объём рынка security audit услуг для SMB (малый и средний бизнес)?
- Сколько компаний ежегодно заказывают security audits?
- Какой типичный бюджет на security у стартапов (seed, Series A)?
- Есть ли данные о росте спроса на security услуги в 2023-2025?

### 2. Ценообразование конкурентов
- Сколько стоят security audits у основных игроков?
  - Традиционные консалтинговые компании
  - Автоматизированные платформы (Snyk, SonarQube, Checkmarx)
  - Фриланс security консультанты (Upwork, Toptal)
  - Bug bounty платформы (HackerOne, Bugcrowd) — средний payout
- Есть ли примеры успешных "express" или "quick" security audit услуг?
- Какой ценовой диапазон для разового аудита 1 репозитория?

### 3. Конкуренты и альтернативы
- Кто предлагает похожие услуги (быстрый аудит, фиксированная цена)?
- Примеры стартапов в этой нише (особенно AI-powered)
- Как позиционируются Snyk, SonarQube для SMB?
- Есть ли white-label security audit решения?

### 4. Каналы привлечения клиентов
- Где стартапы и dev-команды ищут security услуги?
- Какие ключевые слова/запросы используют (для SEO/ads)?
- Работают ли cold outreach, content marketing, partnerships?
- Примеры успешных GTM стратегий security стартапов

### 5. Боли и мотивация покупателей
- Почему компании заказывают security audit? (compliance, инвесторы, breach prevention)
- Какие события триггерят покупку? (fundraising, enterprise deal, incident)
- Что важнее: цена, скорость, глубина, репутация?
- Какие возражения типичны при продаже security услуг?

### 6. Региональная специфика
- Отличия рынка US/EU vs RU/CIS
- Есть ли локальные игроки в России/СНГ?
- Влияние санкций на доступность западных инструментов
- Требования к data sovereignty (self-hosted)

### 7. Тренды и прогнозы
- Как AI меняет рынок security услуг?
- Shift-left security, DevSecOps adoption rates
- Прогнозы роста рынка application security
- Emerging competitors (AI-native security tools)
