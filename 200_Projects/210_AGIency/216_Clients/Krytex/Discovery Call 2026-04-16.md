# Discovery Call — Krytex (16.04.2026)

Демо-звонок с командой Krytex (производитель автохимии: керамические покрытия, антидождь, абразивы, полироли). Показано демо AI чат-бота, обсуждены возможности расширения и каналы подключения.

## Участники

- Сергей (AGIency) + Мария (ассистент, готовила демо)
- Представитель Krytex

## Демо: текущий чат-бот

Бот работает на LLM с одной таблицей "вопросы-ответы" по продукции Krytex.

**Тестовые вопросы и результаты:**

| Вопрос | Результат | Комментарий |
|--------|-----------|-------------|
| Твердость покрытия VORSIC SIC? | 3H по Моосу | Клиент сказал — возможно неверно, надо проверить в таблице |
| Аромат №5 "Наркотический цветок"? | Напоминает Ex Nihilo Fon | Корректно |
| Что делать, если дворники прыгают после MegaGlass? | Удалить антидождь, повторить нанесение | Частично верно — собрал из разных источников |
| AutoGlass Abrasiv — можно на ЛКП? | Нет, абразивное средство / можно для фар | Неточно — смешал данные из нескольких запросов |
| Идёт ли губка в комплекте с Glass Polish? | Не нашёл точного ответа | Ответа нет в базе |
| Инструкция по керамическому молочку VORSIC? | Не нашёл инструкцию | Инструкции в отдельной таблице, не подключена |

**Выводы по демо:**
- Бот понимает разные формулировки (например, "торсик" → VORSIC)
- Галлюцинации снижены промптингом, но не исключены
- Одной таблицы Q&A недостаточно — нужны инструкции, данные о товарах

## Обсуждение: расширение базы знаний

### Вариант 1 — пополнение таблицы Q&A

- Собирать вопросы с маркетплейсов (Ozon, Wildberries)
- Обрабатывать через LLM → привести в единый формат → добавить в таблицу
- Простой подход, но ограниченный — новые вопросы не будут покрыты

### Вариант 2 — полноценная база знаний

- Загрузить инструкции по средствам, описания продуктов, FAQ
- Бот ищет не по таблице, а по базе знаний и составляет ответ
- Менее точно, но покрывает гораздо больше вопросов
- С промптингом "не нашёл — не придумывай" ошибки минимизированы

### Мультиагентная система

- При 6+ таблицах одному боту сложно — нужны несколько специализированных агентов
- Это влияет на стоимость внедрения

## Источники данных Krytex

- **Таблица Q&A** — текущая, вручную наполняемая
- **Вопросы с маркетплейсов** — Ozon, Wildberries (десятки вопросов в день)
- **Отзывы с маркетплейсов** — содержат полезные данные по применению продукции
- **Инструкции по средствам** — отдельная таблица (пока не подключена)
- **API маркетплейсов** — Q&A доступно через API (Ozon — нужен Premium Plus?), отзывы — возможно недоступны

## Функции платформы

- **Интерфейс входящих** — просмотр всех диалогов бота, фильтрация по каналам
- **Human-in-the-loop** — бот генерирует ответ → отправляет в Telegram менеджеру → менеджер проверяет и отправляет клиенту
- **Отзывы как база знаний** — парсить отзывы, добавлять со ссылкой на оригинал

## Стоимость

- **Внедрение** — зависит от количества таблиц/источников и сложности бота
- **Владение (подписка)** — оплата токенов с небольшой наценкой
- **Ориентир**: 100 вопросов ≈ 500 руб, 1000 запросов/мес ≈ до 5000 руб
- **Экономия**: без сохранения истории диалога — дешевле

5. Рассмотреть подключение инструкций по средствам как вторую базу знаний
