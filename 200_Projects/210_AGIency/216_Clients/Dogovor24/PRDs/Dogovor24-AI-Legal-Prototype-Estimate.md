# Оценка трудозатрат: AI Legal Gateway для Dogovor24

> Оценка в часах разработки мультиагентной AI-системы юридических консультаций.
> Стек: Go (Kratos + Ent), PostgreSQL, pgvector, Redis, Docker.

## Допущения

- UI модерации — на стороне D24, в оценку не входит
- Один кодекс, один специализированный агент

## Примечания

1. **Не включено в оценку:**
   - UI модерации (реализация на стороне D24)
   - Юридическая экспертиза НПА (консультации с юристами D24)
   - Управление проектом (PM)
   - Получение и подготовка исторических данных из amicus
2. **Буфер неопределённости** — рекомендуется закладывать +20-30% на непредвиденные задачи (интеграционные проблемы, качество исходных данных НПА, итерации по промптам).
3. **LLM-инфраструктура** — затраты на API-вызовы LLM (GPT-4o / Claude) не включены, это операционные расходы.
