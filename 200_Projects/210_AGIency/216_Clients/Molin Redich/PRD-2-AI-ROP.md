# PRD-2: AI РОП — контроль качества коммуникации сейлзов

**Клиент:** Molin Redich
**Дата:** 2026-03-07
**Статус:** Черновик для оценки

---

## Проблема

Руководитель отдела продаж не может системно контролировать качество коммуникации сейлзов. Звонки, чаты и переписка анализируются вручную и выборочно. Нет объективной оценки по KPI, нет возможности масштабировать контроль на всю команду.

## Решение

Система "AI РОП" — автоматический контроль качества коммуникации менеджеров по продажам:

1. **Транскрибация звонков** — автоматическое преобразование записей в текст
2. **Анализ чатов и почты** — подключение к каналам коммуникации
3. **ИИ-оценка качества** — проверка по скрипту/чек-листу, выявление ошибок
4. **Дашборд KPI** — сводная аналитика по менеджерам и команде

## Функциональные требования

### Источники данных
- Записи телефонных звонков (интеграция с телефонией клиента)
- Чаты с клиентами (мессенджеры, CRM-чат)
- Электронная почта (входящие/исходящие по сделкам)

### Транскрибация звонков
- Автоматическая транскрибация записей (русский язык)
- Speaker diarization (разделение реплик менеджер/клиент)

### ИИ-анализ коммуникации
- Проверка соблюдения скрипта продаж (чек-лист)
- Выявление типовых ошибок (перебивание, отсутствие выявления потребности, нет закрытия на следующий шаг)
- Оценка тональности и вежливости
- Выявление упоминаний конкурентов, возражений
- Формирование оценки по каждому взаимодействию (балл/рейтинг)

### KPI-дашборд
- Средний балл качества по менеджеру / команде / периоду
- Динамика качества (тренды)
- Топ ошибок и зон роста по каждому менеджеру
- Лучшие/худшие звонки для разбора
- Выполнение KPI по объёму звонков, конверсии (если данные доступны)

### Разграничение доступа
- РОП / руководитель: видит всю команду
- Менеджер: видит только свои данные
- Админ: настройка скриптов и чек-листов

## Этапы реализации

| Этап | Описание | Срок |
|------|----------|------|
| 1 | Аудит инфраструктуры клиента, проектирование архитектуры | 4 |
| 2 | Интеграция с телефонией, транскрибация звонков | 16 |
| 3 | Подключение чатов и почты | 16 |
| 4 | ИИ-модуль анализа качества (скрипт, ошибки, оценка) | 40 |
| 5 | Дашборд KPI (веб-интерфейс) | 40 |
| 6 | Тестирование на реальных данных, настройка чек-листов | 40 |

**Общий срок: 6-8 недель**

## Оценка бюджета

**800 000 - 1 500 000 руб.**

Диапазон зависит от:
- Количества источников данных (телефония + чаты + почта vs только звонки)
- Сложности интеграции с телефонией клиента
- Требований к дашборду (простой vs расширенный)
- Количества менеджеров и объёма данных
- Стоимости API для транскрибации (зависит от объёма звонков)

### Операционные расходы (ежемесячно)
- Транскрибация: зависит от объёма (ориентир: 5-15 тыс. руб/мес при 50-200 звонках/день)
- LLM API для анализа: 3-10 тыс. руб/мес
- Серверная инфраструктура: зависит от размещения

## Что НЕ входит

- CRM-система (предполагается, что у клиента уже есть)
- Телефония (используется существующая)
- Обучение менеджеров (только настройка системы и передача руководителю)
- Real-time подсказки менеджеру во время звонка (может быть расширением)
- Интеграция с системой мотивации / расчёт бонусов
