# AGIents.pro — Платформа для создания ИИ-агентов

B2B no-code платформа для создания ИИ-агентов.

## Проблема

Бизнесам нужны кастомные ИИ-агенты, но нет технических ресурсов для разработки.

## Целевая аудитория

- B2B клиенты
- Компании без технических ресурсов для разработки
- Бизнесы, которым нужна автоматизация с ИИ

## Ключевые особенности

- No-code подход
- Кастомизация под клиента
- B2B модель
- Соло-проект (полный контроль)

## Структура папок

- `221_Roadmaps/` — дорожные карты развития
- `222_Research/` — исследования и материалы
- `223_PRDs/` — спецификации продукта
- `224_Leads/` — лиды и потенциальные клиенты
- `225_BizDev/` — бизнес-развитие
