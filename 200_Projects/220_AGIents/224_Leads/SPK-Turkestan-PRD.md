# Интеллектуальный ассистент для АО «СПК «Түркістан»

## Описание продукта

Единая интеллектуальная платформа, которая позволяет сотрудникам и руководству АО «СПК «Түркістан» мгновенно находить нужную информацию в корпоративных документах, получать готовые выжимки и справки, а также обеспечивает консультационную поддержку внешних пользователей через сайт организации.

Платформа работает как «цифровой эксперт», который знает все внутренние документы компании и может за секунды найти нужный ответ — с указанием конкретного источника.

---

## Проблемы, которые решает система

### Для руководства
- **Долгий поиск информации.** Решения, регламенты, протоколы накапливаются годами. Найти нужный пункт в архиве — задача на часы
- **Риск устаревших данных.** Документы обновляются, но сотрудники могут опираться на старые редакции
- **Отсутствие единой точки доступа.** Документы хранятся в разных местах — почта, папки, архивы, у отдельных сотрудников

### Для сотрудников
- **Рутинная подготовка справок и записок.** Сбор информации из нескольких источников для подготовки одного документа
- **Зависимость от коллег.** «Кто знает, где лежит тот документ?» — частый вопрос
- **Невозможность быстро сравнить редакции.** Что изменилось в регламенте за последний год?

### Для внешних пользователей
- **Отсутствие оперативной консультации.** Граждане и партнёры не могут быстро получить ответы по деятельности организации через сайт

## Состав решения

### Модуль 1: Корпоративный ИИ-ассистент

Закрытая система для сотрудников организации. Доступна через внутренний портал.

**Возможности:**

- **Умный поиск по документам.** Находит ответы по смыслу, а не только по ключевым словам. Если вы спросите «что изменилось в бюджетной политике», система найдёт все релевантные документы, даже если в них не используется слово «бюджетная политика»
- **Ответы с источниками.** Каждый ответ сопровождается ссылкой на конкретный документ, раздел и страницу. Можно сразу открыть и проверить
- **Сравнение версий документов.** «Что изменилось в положении о закупках между 2023 и 2025 годом?» — система покажет разницу
- **Сводки и резюме.** Длинный документ на 50 страниц — система подготовит краткое резюме с ключевыми пунктами за секунды
- **Помощь в подготовке документов.** Черновики справок, служебных записок, ответов на запросы — на основе внутренних шаблонов и базы знаний
- **Работа с внешними источниками.** По решению администратора система может обращаться к официальным порталам РК (НПА, государственные реестры) для дополнения ответов актуальной нормативной информацией

**Разграничение доступа:**

Каждый сотрудник видит только те документы, к которым имеет допуск. Руководство — полный доступ. Юристы — правовые базы и разрешённые разделы. Сотрудники подразделений — документы своего подразделения и общая база.

### Модуль 2: Публичный ИИ-консультант

Встроенный в сайт spkturkistan.kz чат-бот для внешних пользователей.

**Возможности:**

- Ответы на вопросы о деятельности организации, проектах, услугах
- Навигация по сайту и помощь в поиске информации
- Работает круглосуточно, без выходных
- Поддержка казахского и русского языков

**Что это даёт:**
- Снижение нагрузки на сотрудников от типовых обращений
- Повышение качества обслуживания граждан и партнёров
- Современный имидж организации

## Какие документы обрабатывает система

- Уставные документы, положения, регламенты
- Приказы, распоряжения, протоколы заседаний
- Проектная документация, отчёты
- Инвестиционные и финансовые документы
- Нормативные правовые акты РК (внешние источники)

**Форматы:** PDF, Word, Excel, PowerPoint, HTML-страницы, сканированные документы (автоматическое распознавание текста)

## Безопасность и конфиденциальность

- **Закрытый контур.** Система разворачивается на инфраструктуре заказчика. Данные не покидают периметр организации
- **Контроль доступа.** Каждый пользователь видит только разрешённые документы
- **Журнал действий.** Все запросы и ответы фиксируются для аудита
- **Политика источников.** Система отвечает только на основе проверенных документов — без «выдумывания» информации. Каждый ответ подкреплён ссылкой
- **Контролируемый внешний доступ.** Обращение к внешним источникам (порталы РК) — только по утверждённому списку, с фиксацией в журнале

Окончательный вариант выбирается совместно с заказчиком с учётом требований информационной безопасности.

## Этапы реализации

### Этап 1. Аналитика и согласование (1 месяц)
- Аудит документов: объём, форматы, уровни конфиденциальности
- Определение ролей и прав доступа
- Согласование требований к безопасности
- Выбор 1-2 подразделений для пилотного запуска
- Утверждение плана проекта

**Результат:** Согласованное техническое задание и план пилота

### Этап 2. Подготовка и загрузка данных (1 месяц)
- Сбор и систематизация документов
- Распознавание сканированных документов
- Загрузка в систему, проверка качества
- Формирование тестовых вопросов для проверки

**Результат:** Наполненная база знаний, готовая к тестированию

### Этап 3. Настройка и тестирование (1-2 месяца)
- Развёртывание системы на инфраструктуре заказчика
- Настройка ролей и доступов
- Пилотный запуск в 1-2 подразделениях
- Сбор обратной связи, корректировки

**Результат:** Работающая система, прошедшая пилотное тестирование

### Этап 4. Запуск и обучение (1 месяц)
- Ввод в эксплуатацию для всех подразделений
- Запуск публичного консультанта на сайте
- Обучение пользователей и администраторов
- Передача документации

**Результат:** Система в промышленной эксплуатации

**Общий срок: 4-5 месяцев**

## Почему мы

- **Готовая платформа.** Мы не пишем систему с нуля — у нас есть работающая платформа для создания ИИ-ассистентов, проверенная на реальных проектах. Это сокращает сроки и снижает риски
- **Опыт внедрения.** Наши решения уже работают у корпоративных клиентов — от консультационных ИИ-ассистентов до систем обработки обращений
- **Гибкость.** Система изначально спроектирована для работы с разными задачами. Корпоративный ассистент, публичный консультант, специализированные агенты для отдельных подразделений — всё на одной платформе
- **Масштабируемость.** Система рассчитана на рост: увеличение объёма документов, количества пользователей, добавление новых функций — без переписывания с нуля
- **Поддержка казахского языка.** Система работает на казахском и русском языках
