## Структура проекта (стандарт)

Каждый проект содержит подпапки по схеме `2X1`, `2X2`... где X — номер проекта:

- `Roadmaps/` — дорожные карты
- `Research/` — исследования
- `PRDs/` — спецификации продукта
- `Leads/` / `Clients/` — лиды и клиенты
- `Case_Studies/` — кейсы
