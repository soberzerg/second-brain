# Создание призывов к действию CTA

**Категория:** Копирайтинг
**Назначение:** Генерация убедительных призывов к действию (CTA) для различных каналов -- от посадочных страниц и email-рассылок до социальных сетей и видео.

## Текст промпта

### Промпты

- Предложи 5 разных призывов к действию для этого текста: [скопируйте и вставьте текст].
- Предложи лучший призыв к действию для [домашней/продуктовой/о нас] страницы для [типа бизнеса].
- Предложи 5 способов, как я могу убедительнее призвать людей к [призыв к действию].
- Придумай сильный призыв к действию, чтобы посетители моей посадочной страницы подписались на мой информационный бюллетень.
- Создай убедительный призыв к действию для моего поста в социальных сетях, который продвигает мой новый продукт.
- Предложи мощный призыв к действию для моей электронной рассылки, который побудит подписчиков к покупке моего нового курса.
- Придумай убедительный призыв к действию для моей рекламной кампании на Facebook, чтобы люди записались на дискавери-звонок со мной.
- Создай сильный призыв к действию для моего веб-сайта, чтобы посетители скачали мою бесплатную электронную книгу.
- Придумай ясный и эффективный призыв к действию для моего видео на YouTube, чтобы зрители подписались на мой канал.
- Напиши убедительный призыв к действию для моей страницы регистрации вебинара, чтобы люди зарегистрировались.
- Предложи заманчивый призыв к действию для моего вступительного ролика на новый подкаст, чтобы слушатели следили за будущими эпизодами.
- Создай убедительный призыв к действию для моей страницы продаж, чтобы посетители купили мой онлайн-курс.
- Помоги мне придумать сильный призыв к действию для моего всплывающего окна перед выходом с сайта, чтобы посетители присоединились к моему списку рассылки перед тем, как уйдут.
