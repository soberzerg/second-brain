# Написание квиза для рекламы

**Категория:** Маркетинг
**Назначение:** Создание цепляющего квиза из 5 вопросов с вариантами ответов для рекламных кампаний в социальных сетях с целью сбора контактов.

## Текст промпта

"Хочу сделать квиз для рекламы. Напиши 5 цепляющих вопросов и 4 варианта ответов на каждый вопрос для квиза, который заполняют пользователи кликнув по рекламе из социальных сетей на тему (заголовок). Задача квиза - чтобы пользователь в конце оставил контакты для (следующий шаг)"
