## Ключевые фреймворки

### GIST Framework (Google)

**Использование:**
- Planning AGIents MVP
- Валидация Ideas перед разработкой
- Приоритизация features

**Компоненты:**
- **G**oals - OKRs, North Star Metric
- **I**deas - множество подходов для тестирования
- **S**teps - валидация через Wide Spectrum
- **T**asks - конкретная реализация

### BATON Framework

**Использование:**
- Бизнес-планирование AGIents
- Стратегия AGIency

**Компоненты:**
- **B**usiness - ценность
- **A**udience - привлечение внимания (блог, соцсети)
- **T**ribe - захват email, community
- **O**ffer - монетизация (tripwire → core → premium)
- **N**etwork - масштабирование

### ICE Scoring

**Использование:**
- Приоритизация features AGIents
- Выбор Ideas для валидации

**Компоненты:**
- **I**mpact - влияние на метрики
- **C**onfidence - уверенность в impact (Confidence Meter)
- **E**ase - легкость реализации
