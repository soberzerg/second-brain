# Выжимка: Build Your First AI Marketing Team (Claude Skills Tutorial)

**Источник:** https://youtu.be/X8afcX2s2Mo?si=RXLSvLHks0yQimiz
**Дата:** 2026-02-23

## Основная тема

Как создать и использовать пользовательские Claude Skills для построения AI-маркетинговой команды: автоматизация рутинных задач, оркестрация навыков и упаковка в переносимые плагины.

## Ключевые тезисы

1. **Claude Skills — это экспертиза, а не просто действия** - Истинная мощь Claude Agents заключается не в способности действовать, а в "экспертизе", которую они используют. Skills позволяют обучать AI конкретным методам работы, а не просто давать ему инструменты.

2. **Skills Library как хранилище маркетинговых знаний** - Библиотека навыков содержит SOPs, проверенные фреймворки, стандарты бренда и ключевую документацию компании. Это делает всю накопленную экспертизу доступной для AI-агентов.

3. **Превращение SOP в многоразовые Skills** - Существующие внутренние документы Standard Operating Procedure можно конвертировать в Claude Skills, которые будут следовать чётко определённым процедурам. Пример: Skill для маркетинговых исследований, использующий Perplexity MCP tool для сбора данных и формирования структурированного отчёта.

4. **Контекстные файлы и `CLAUDE.md` — фундамент** - Для каждого проекта нужна файловая структура: папки `context` (brand voice, продукт, стратегия), `data` (данные), `SOP` (процедуры). Файл `CLAUDE.md` в корне проекта — пользовательские инструкции для Claude о навигации и использовании контекста.

5. **"Show, Don't Just Tell" при обучении AI** - Для качественных результатов нужны не только описания, но и лучшие примеры (`examples` folder) и референсы, из которых Claude извлекает паттерны и best practices.

6. **Оркестрация навыков** - Claude может самостоятельно определить наиболее эффективный способ использования нескольких навыков вместе. Он анализирует запрос, релевантные навыки и контекстные файлы, создаёт план выполнения и распределяет работу.

7. **Subagents Mode (90% кейсов)** - Claude выступает "Team Lead", распределяя задачи между субагентами (навыками). Каждый субагент отчитывается только тимлиду. Оптимален для независимых задач, потребляет меньше токенов.

8. **Agent Teams Mode (экспериментальный)** - Агенты имеют общий список задач и могут напрямую взаимодействовать между собой, обмениваясь обратной связью. Полезен для сложных рабочих процессов с высокой степенью координации, но потребляет больше токенов.

9. **ИИ берёт на себя "heavy lifting", но не стратегию** - AI эффективен для мозгового штурма, анализа рынка, конкурентов и сбора данных. Но без человеческой стратегии и аутентичности результат — лишь "automating average".

10. **Claude Plugins для переносимости** - Набор кастомных навыков упаковывается в ZIP-архив (Bundled Package). Это позволяет импортировать библиотеку навыков в другие проекты Claude или Claude Cowork — мгновенная готовность для новых брендов.

## Полезные инсайты

- **Процесс важнее инструмента** - Хорошо определённый workflow, встроенный в навык, важнее самого инструмента исследования. Навык без процесса — просто обёртка.

- **"The Loop Marketing Framework" от HubSpot** - Маркетинг — не линейная воронка, а непрерывный цикл: Express → Tailor → Amplify → Evolve. Человеческая стратегия и ИИ работают вместе для постоянной оптимизации.

- **Claude как координатор команды** - В режиме Agent Teams Claude выступает тимлидом, самостоятельно распределяя задачи между специализированными навыками и координируя их для достижения комплексных целей.

- **Экономия на подготовительной работе** - AI-навыки сокращают время на сбор информации, форматирование, черновики и отчёты, освобождая маркетологов для стратегического мышления.

## Практические рекомендации

- [ ] **Оцифровать свои SOPs** - Стандартизировать внутренние маркетинговые процессы для создания кастомных Claude Skills
- [ ] **Создать контекстную библиотеку** - Для каждого проекта/бренда структурированная папка `context` с brand guidelines, voice, product offering, growth marketing
- [ ] **Использовать `skill-creator`** - Официальный инструмент Anthropic для преобразования описаний SOP и фреймворков в полноценные Claude Skills
- [ ] **Включать примеры в навыки** - При создании навыков всегда добавлять качественные примеры и референсы, а не только инструкции
- [ ] **Начинать с Subagents Mode** - Для большинства задач, разбиваемых на независимые части, субагенты эффективнее
- [ ] **Упаковывать навыки в плагины** - Для переносимости и мгновенного развёртывания в новых проектах

## Упомянутые ресурсы

- **Claude Skills** (функционал) - Основа демонстрации, кастомные навыки для Claude
- **Claude Code Desktop** (IDE) - Среда разработки от Anthropic для создания и управления навыками
- **Claude Cowork** (инструмент) - Нетехническая версия Claude Code для офисной работы
- **`skill-creator`** (инструмент Anthropic) - Официальный генератор навыков
- **Official Skills Repository** (репозиторий) - Репозиторий с официальными навыками Anthropic
- **Perplexity MCP tool** (инструмент) - Для глубокого исследования и поиска информации
- **Google Nano Banana model** (модель) - Для генерации изображений/креативов в Claude
- **HubSpot - "The Loop Marketing Landscape Report"** (отчёт) - Бесплатный отчёт по опросу 1800+ маркетологов об адаптации к ИИ
- **"AI Growth Systems for Marketing"** (программа) - Платное сообщество автора видео о системах роста с ИИ
