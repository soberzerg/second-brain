# YouTube Выжимки

Автоматически созданные выжимки из YouTube видео через skill youtube-summary.

## Использование

Просто отправь YouTube URL в чат с Claude, и выжимка создастся автоматически.

## Формат файлов

`YYYY-MM-DD - [Название видео].md`

## Теги

Все файлы автоматически помечаются тегами:
- `youtube` - всегда
- `выжимка` - всегда
- По теме видео (ai, бизнес, разработка, и т.д.)
