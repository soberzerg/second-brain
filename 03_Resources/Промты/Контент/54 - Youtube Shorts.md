# Youtube Shorts

**Категория:** Контент
**Назначение:** Создание сценариев для коротких видео YouTube Shorts -- от рекламных роликов и видеовизиток до технических инструкций и видеопрезентаций.

## Текст промпта

Помимо работы с полноценными видео на YouTube, ChatGPT может помочь спроектировать сценарий для яркого Shorts. Нейросеть может исследовать актуальные тренды видеохостинга.

### Основной промпт

Сгенерируй сценарий для 30-секундного рекламного ролика, продвигающего наш новый [продукт].

**Пример:** Сгенерируй сценарий для 30-секундного рекламного ролика, продвигающего наш новый смартфон с широкоугольной камерой и 256 Гб памяти.

### Другие промпты

1. Напиши сценарий для 25-секундного видео, в котором зрители увидят, как наш новый [продукт] поможет им в повседневной жизни.
2. Проанализируй самые популярные темы для YouTube Shorts и предложи свой вариант интересного ролика на 60 секунд.
3. Создай забавный сценарий для Shorts на YouTube, чтобы подписчики захотели отправлять больше положительных реакций.
4. Сгенерируй короткий рекламный ролик для YouTube Shorts о нашем [продукт]. Укажи, что его главные свойства должны отображаться в течение видео на экране.
5. Напиши сценарий для 30-секундного видео, в котором зрители увидят, как наш новый [продукт] помогает им решать сложные задачи на работе.
6. Создайте краткий сценарий для YouTube Shorts, который демонстрирует, как наш новый [продукт] может помочь зрителям получить лучшие результаты в спорте или фитнесе.
7. Напишите сценарий для короткого рекламного ролика для YouTube Shorts, который демонстрирует, как наш новый [продукт] может помочь улучшить жизнь и поддерживать связь с друзьями и близкими.
8. Напиши сценарий для 20-секундного видео, в котором зрители увидят, как наш новый [продукт] улучшит их работу или учебу.
9. Придумай 5 разных сюжетов для видео YouTube Shorts на 30 секунд по [тема].
10. Создай забавный и неожиданный сценарий для Shorts на YouTube, в котором наш новый [продукт] будет главным героем и поможет герою преодолеть проблему.
11. Создай вдохновляющий сценарий для Shorts на YouTube, в котором люди будут использовать наш новый [продукт], чтобы захватывать и делиться своими приключениями и впечатлениями в социальных сетях.
12. Напиши сценарий в формате визитки для YouTube Shorts молодого блогера, который ведет канал на [тема]. Добавь призыв переходить на канал блогера, подписываться и смотреть его ролики.
13. Напиши сценарий в формате технической инструкции для [продукт] на YouTube Shorts.
14. Напиши сценарий для [продукт] для Shorts на YouTube. Покажите, как наш [продукт] может быть применен в разных сферах жизни.
15. Создай остроумный сценарий для YouTube Shorts на тему [тема], который будет включать наш новый [продукт] в неожиданных ролях.
16. Создай забавный сценарий для Shorts на YouTube, который покажет зрителям, как наш новый [продукт] может помочь им расслабиться и развеять негативные эмоции.
17. Напиши сценарий для 20-секундного видео, в котором зрители увидят, как наш новый [продукт] поможет им стать более организованными и эффективными в их повседневных делах.
18. Напиши сценарий для 60-секундной видеопрезентации [продукт] на YouTube Shorts. Добавь в конце видео мощный призыв заказать [продукт] прямо сейчас, переходя по ссылке в описании.
19. Создай интересный сюжет для YouTube Shorts на 45 секунд, в котором герой использует наш новый [продукт] для достижения своей цели.
