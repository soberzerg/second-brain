# Заголовок для Youtube

**Категория:** Контент
**Назначение:** Генерация кликбейтных и привлекательных заголовков для видео на YouTube, подкастов, презентаций и рекламных объявлений.

## Текст промпта

### Промпты

1. Предложи 10 вариантов кликбейтных заголовков объемом не более 50 символов для канала на [тема] для видео [название].
2. Придумай 5 привлекающих внимание заголовков для этого текста: [скопируйте и вставьте сценарий видео].
3. Напиши 5 привлекающих внимание заголовков для видео на YouTube на [тема].
4. Приведи примеры броских заголовков для видео на YouTube на [тема], которые заставят людей щелкнуть и посмотреть видео. Сделай их в [стиле].
5. Проанализируй самые популярные видео на YouTube и предложи 5 своих вариантов заголовков к ним.
6. Придумай интригующие заголовки для видео на YouTube по [тема].
7. Предложи 5 идей для создания заголовков видео на YouTube, которые увеличивают количество подписчиков и лайков.
8. Расскажи, как использовать ключевые слова в заголовке видео на YouTube, чтобы привлечь больше зрителей.
9. Приведи примеры заголовков для [тип контента] на [тема], которые будут вызывать интерес у целевой аудитории.
10. Предложи 5 вариантов заголовков для презентации на [тема], которые заинтересуют зрителей.
11. Придумай интригующие заголовки для подкаста на [тема], которые заставят людей прослушать эпизод.
12. Напиши 5 заголовков для рекламных объявлений на [тема], которые привлекут внимание потенциальных клиентов.
13. Приведи примеры броских заголовков для статей на [тема], которые заставят людей начать чтение. Сделай их в [стиле].
14. Создай заголовок для видео на [тема], содержащий [ключевые слова].
15. Напиши 10 длинных кликбейтных заголовков для видео на YouTube на [тема], которые будут выделяться среди других по такому же поисковому запросу.
16. Приведи примеры убедительных заголовков для баннеров на [тема], которые будут использоваться в видео на YouTube.
17. Как написать заголовок для своего первого видео на YouTube на [тема]? Предложи 5 вариантов, которые будут привлекать внимание зрителей.
18. Придумай 5 эффективных заголовков для видео на YouTube на тему личностного роста и саморазвития.
19. Как использовать смешанные заголовки в видео на YouTube, чтобы привлечь больше зрителей? Предложите 5 примеров.
20. Как использовать заголовок видео на YouTube, чтобы увеличить просмотры и зрительскую активность? Приведите 5 примеров.
