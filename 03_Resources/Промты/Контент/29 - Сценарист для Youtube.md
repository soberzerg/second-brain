# Сценарист для Youtube

**Категория:** Контент
**Назначение:** Набор промптов для создания сценариев видео на YouTube -- от генерации идей и написания текста до преобразования готового контента в формат видеосценария.

## Текст промпта

Если вы ведете канал на YouTube, то знаете, что процесс подготовки захватывающего ролика может занимать не один день. ChatGPT может решить целый ряд трудностей, связанных с креативным сценарием и описанием для вашего видео. Используйте специальные промпты и получите уникальный текст по вашему запросу, кликбейтные заголовки и SEO-описания под видео.

### Промпты для создания сценария видео

У вас уже есть идеи и даже текст, но вы не знаете, как оформить его в виде хорошего сценария? Используйте следующие команды, чтобы создать полноценный сценарий для видео на основе готового текста или сгенерировать текст для видео с нуля.

#### 1. Из поста в сценарий

Напиши пост для [сайт/блог/страница] по [тема], упоминая эти преимущества [объект]:
- [Преимущество 1].
- [Преимущество 2].
- [Преимущество 3].

Теперь преобразуй получившийся текст в сценарий для видео на YouTube.

**Пример использования:**
Напиши пост от лица профессионального диетолога с 10-летним стажем для блога о диетологии по теме "В чем вред интервального голодания?", упоминая эти преимущества блога:
- В блоге рассказывают о современных тенденциях в диетологии и здоровом питании с научной точки зрения.
- При оформлении недорогой платной подписки на блог пользователи получают в подарок гайд по здоровому питанию и книгу ПП-рецептов.
- Вместе с автором блога сотрудничают лучшие медицинские специалисты, фитнес-инструкторы и люди, которым удалось наладить режим питания и избавиться от тяжелых заболеваний, таких как ожирение, пищевые аллергии, расстройства ЖКТ и т.п.

*Второй запрос:* Теперь преобразуй получившийся текст в сценарий для видео на YouTube.

#### 2. Базовые промпты для сценариев

2. Напиши [тип текста] по [тема] для [аудитория] в [стиле], укажи [дополнительная информация]. Преобразуй получившийся текст в сценарий для YouTube.

3. Напиши очень увлекательный сценарий для видео на YouTube по [тема].

4. Придумай 10 актуальных тем для видео на YouTube канале на [тема]. Выбери одну из тем и предложи сценарий в [структура] и [стиль], а также включи в конце видео [информация о канале].

5. Создай сценарий для видео на YouTube в структуре FAQ для нового [продукт]. Опиши [уникальные свойства] [продукт], расскажи о производстве/создании [продукт], познакомь зрителей с создателями [вставить дополнительную информацию].

6. Напиши рекламное предложение для [целевая аудитория] о [продукт], который решает [проблемы]. Используй прием сторителлинга -- расскажи историю о трех разных героях, которые приобрели [продукт] и решили ряд [проблемы]. Добавь в конце мощный call-to-action и просьбу переходить по ссылке, чтобы купить [продукт]. Преобразуй получившийся текст в формат сценария для видео на YouTube.

7. Предложи 10 продающих вариантов заголовков по [тема]. Добавь в заголовки [ключевые слова]. Напиши на самую популярную из этих тем небольшую статью на [количество слов] для блога. Составь сценарий для видео на YouTube на основе текста выше, добавив в него 5 разных интересных фактов по [тема].

8. Предложи 10 интересных фактов по [тема]. Преобразуй их в сценарий для видео на YouTube.

9. Напиши интригующий сценарий для видео на YouTube по [тема], добавь в конце видео розыгрыш [приз] среди самых активных комментаторов данного видео.

10. Предложи сценарий для короткого видео на YouTube длительностью не более 5 минут по [тема].

#### 3. Продвинутые промпты

11. Включи в сценарий для видео полезный совет для [целевая аудитория] по [тема]. Используй [количество] примеров, чтобы проиллюстрировать свой совет и дать практические советы для реализации.

12. Создай сценарий для видео на YouTube, в котором ты демонстрируешь свой процесс [хобби]. Расскажи о том, как ты начал заниматься своим хобби, дай советы для новичков и покажи свои лучшие работы.

13. Напиши веселый сценарий для видео на YouTube, в котором ты проводишь интервью с [известный человек] по [тема]. Поделись их историей, профессиональными советами и вдохновляющими историями успеха.

14. Создай сценарий для видео на YouTube, в котором ты показываешь, как создать [продукт] с нуля. Расскажи о необходимых материалах и инструментах, объясни процесс и дай полезные советы для достижения наилучших результатов.

15. Напиши сценарий для влога на YouTube, в котором ты покажешь, как проводишь свой день. Опиши свои [любимые занятия], [места], которые посещаешь, и [еда], которую ешь. Расскажи зрителям, почему тебе так нравится проводить время, и дай им несколько советов, как сделать свой день интереснее.

16. Создай сценарий для видео на YouTube о путешествии в [место]. Расскажи зрителям, какие достопримечательности они могут посетить, какие места стоит посмотреть, и какие местные блюда нужно попробовать. Дай зрителям несколько советов, как сэкономить деньги на поездке и как сделать свое путешествие незабываемым.

17. Напиши динамичный сценарий для видео на YouTube по [тема] с использованием интерактивных элементов, таких как голосования и опросы.

18. Создай образовательный сценарий для видео на YouTube по [тема] с использованием анимации и графики.

19. Напиши сценарий в формате расследования на [тема] для любителей [предмет/тема]. Представь аудитории новые факты и неожиданные открытия в [стиле]. Укажи, что тебя вдохновило на данное расследование.

20. Составь подборку советов и хитростей для [аудитория] по теме [тема]. Оформи в [стиле] и добавь некоторые свои личные советы. Укажи, как тебе удалось накопить такой богатый опыт в данной теме.
