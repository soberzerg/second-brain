## 1. Резюме стратегии

Май-2026 — это «месяц консолидации» в цикле Q2-плана. Главная задача — превратить накопленный за апрель материал (CLAUDE.md, harness > model, Krytex deal, AgentOS Phase 2 PRD, Instagram-карусели) в **связанную воронку**: Threads/Instagram цепляют новых читателей → Telegram даёт глубину и ведёт через два маршрута («МОЗГ» / «ДИАГНОЗ») к лид-магнитам → лид-магниты прогревают на трипваеры (5К/10К) и аудиты (30К/50К). Параллельно блог должен накачивать **buildinpublic** вокруг AGIents.pro: апрель закрыли с 210 подписчиками в Telegram и 297 в Threads, цель Q2 — 1.5–2.5K подписчиков и 50–100 лидов, поэтому май фокусируется на росте верха воронки и одновременно на проверке конверсии в лид-магнит.

Контент по-прежнему строится по методу **«Диагностирующего блога»** ([[Руководство - Диагностирующий блог]]): не «лечить» бесплатно, а диагностировать, ставить открытые петли и вести по маршруту. 5 пилларов из [[Система контента - AISobolev]] перетасованы под месяц: каждая неделя имеет один доминирующий пиллар, чтобы аудитория считывала прогресс «диагноз → кейсы → buildinpublic → позиция».

## 3. ICP и аудитории

### Основная ЦА (80% контента) — Фаундеры

См. полный портрет: [[Нейроанализ ЦА - Фаундеры]].

- **Ситуация:** владелец сервисного бизнеса / e-commerce / инфобиза, команда 3–10 чел., оборот 3–50 млн ₽/год.
- **Триггер:** «Знаю, что ИИ может помочь, но не понимаю, с чего начать конкретно в моём бизнесе».
- **Глубокий страх:** «Конкуренты с ИИ обгонят и заберут моих клиентов».
- **Маршрут воронки:** **«МОЗГ»** → «Второй мозг фаундера» (GitHub-шаблон + CLAUDE.md) → трипваер 10К (настройка) → ИИ-аудит 30К → внедрение от 150К → ретейнер.

### Вторичная ЦА (20% контента) — Директора МСБ

См. полный портрет: [[Нейроанализ ЦА - Директора МСБ]].

- **Ситуация:** директор / собственник, 32–52 года, команда 10–300 чел., оборот 10–500 млн ₽/год.
- **Триггер:** «Все говорят про ИИ, а я не понимаю, как применить к моему бизнесу».
- **Маршрут воронки:** **«ДИАГНОЗ»** → бот «3 вопроса» → трипваер 5К (экспресс-разбор) → честный аудит 50К → внедрение → ретейнер.

### Вспомогательный сегмент — вайбкодеры/разработчики

Высокий органический охват в Threads/Instagram (CLAUDE.md, Claude Code skills, Ralph Loop), не основной для монетизации, но даёт reach и социальное доказательство. План: ≤25% контента, использовать как «верхнюю воронку», уводить в Telegram через лид-магнит «Git для вайбкодера» ([[Лид-магнит - Git для вайбкодера]]).

## 8. Repurposing — как один материал становится 4–5 единицами контента

### Принцип

> Один центральный материал на неделе → пересборка под формат и аудиторию каждой площадки. Не копипаст. См. правило [[2026-03-16 - Threads Strategy for AISobolev]] §3.4.

### Поток на одну неделю

```
TG-понедельник (диагностический пост, 1500 символов)
    │
    ├──> Threads-понедельник: вытаскиваем главный удар (300 символов) + ссылка на TG в био
    │
    ├──> X (EN)-вторник: thread из 4–6 твитов на английском, адаптация без англицизмов в RU
    │
    ├──> Instagram-карусель-пятница (1 раз в неделю): если пост был «список / чек-лист / сравнение» — превращаем в карусель из 7–12 слайдов через готовый skill `social-content`
    │
    └──> LinkedIn-четверг (если это лучший пост недели): развёрнутая B2B-версия с акцентом на ROI и TCO

TG-четверг (мнение / позиция, 800 символов)
    │
    ├──> Threads-четверг: одна резкая строка + расшифровка
    │
    └──> LinkedIn (если RU-пост лёг идеально на B2B-аудиторию)

TG-пятница (кейс с цифрами, 1500–2000 символов)
    │
    ├──> Threads-пятница: «было / стало» в 200 символах
    │
    └──> X (EN)-пятница: case-study thread с цифрами
```

### Правила переиспользования

1. **Не «обрезать», а пересобрать.** Threads — другая мысль, другой ритм. Ритм Telegram — длинный заход, ритм Threads — удар + точка.
2. **Карусель ≠ скриншот поста.** Карусель — отдельный креатив с визуальным ритмом. Использовать готовый пайплайн (slidev / nano-banana).
3. **EN-адаптация — не Google Translate.** Заменять русские примеры на универсальные (например, упоминание «ЦА — фаундер из СНГ» убрать, оставить «solo founder»).
4. **Каждое переиспользование — мост обратно в Telegram.** Нет одиночных площадок: Threads/X/LinkedIn/Instagram → Telegram → лид-магнит.

### Бюджет времени (целевой)

| Активность | Часов/неделю |
|-----------|--------------|
| Написание Telegram-постов (4 шт × 30–45 мин с использованием skill `social-content`) | 2.0–3.0 |
| Threads — пересборка (5 шт × 10 мин) | 0.8–1.0 |
| X / LinkedIn — адаптация | 0.5–0.8 |
| Instagram-карусель (1 в неделю) | 1.5–2.0 |
| Мониторинг + ответы / комменты | 1.0–1.5 |
| **Итого/неделя** | **5.8–8.3** |

Это укладывается в бюджет «5–7ч/нед на блог» из [[2026 Yearly Plan]].
