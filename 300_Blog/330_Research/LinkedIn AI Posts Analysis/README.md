## Паттерны успешных постов

### По формату
- **Contrarian take** — самый мощный формат (Mollick, Shi, Sheikh)
- **Story + data** — высокий comment ratio (Rashidi: 250 комментов)
- **Step-by-step tutorial** — максимум комментов через CTA (Russo: 503)
- **Structured breakdown** — хорошие шеры (Grennan: 215 шеров)
- **Research-backed** — максимум лайков (Mollick: 5,384)

### По хукам
- Конкретные цифры в первых строках ("776 professionals", "1,000 jobs", "150+ companies")
- Contrarian opening ("AI war is not about intelligence", "AI competition is weird")
- Провокационное обещание ("We tried replacing 1000 human jobs")
- Прямое ценностное предложение ("This is the best way to create your AI clone")

### По тезисам
- **Anti-hype** работает лучше хайпа (Shi: "only 15% solvable", Rashidi: "AI was wrong")
- **Framework/System thinking** вовлекает enterprise аудиторию (Kriaris, Sheikh)
- **Behind-the-scenes** раскрытие скрытой логики (Grennan: "инвестиции = land grabs")
- **Research + personal take** = авторитет + уникальность (Mollick)

### Engagement metrics insights
- **Высокий comment ratio** = история + эмоция (Sol Rashidi: 26% comment/like ratio)
- **Высокие шеры** = полезный контент + expertise (Andrew Ng: 6.6% share/like ratio)
- **CTA-driven comments** = tutorial + "comment X to get Y" (Michele Russo: 60% comment/like ratio)
