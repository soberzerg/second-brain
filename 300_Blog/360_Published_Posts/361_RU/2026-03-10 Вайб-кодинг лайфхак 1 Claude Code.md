# Вайб-кодинг для бизнеса. Лайфхак №1

**Платформа:** Telegram
**Статус:** Черновик
**Связанный проект:** [[210_AGIency]]
**Источники:** [SWE-bench](https://www.swebench.com), [Morph AI — 15 Agent Comparison](https://morphllm.com/ai-coding-agent)

---

80.9% задач решает автоматически. Не программист. Инструмент!

```txt
#вайбкодинг лайфхак №1

Все знают (ну или, скажем так, это общеизвестно), что Claude — лучшие модели для кодинга. Есть такой бенчмарк для моделей — SWE Bench (и его разновидности). Это главный мировой бенчмарк по кодингу, и модели Claude уже долгое время занимают в нём топ-1. Особенность этого бенчмарка в том, что он оценивает только работу самой модели, не оценивая работу агента, но при этом все модели ставятся в равные условия (что тоже важно для бенчмаркинга).

Но вот что интересно: дело не только в модели.

Есть исследование от Morph AI, которое переворачивает голову: одну и ту же модель (Claude Opus 4.5) запускали в разных инструментах — Claude Code, Cursor, Antigravity, Codex и Devin. Разница со вторым местом — 34 задачи из 731. Claude Code решил их. Остальные — нет.

Аналогия простая: это как нанять сильного специалиста и посадить его в токсичную команду с кривыми процессами. Человек тот же. Результат — другой.

Вы спросите, почему так? Модель-то одна и та же!

Да, именно. Разница в "просто обёртке":

1. Системном промте агента (это базовая инструкция, которая вшита на уровне приложения)
2. Используемых им инструментах (а тут секрет в том, что Anthropic тренировали Claude работе именно с этими инструментами)

Кроме этого, Claude Code является де-факто стандартом для AI-кодинга, и первым вводит улучшения, которые постепенно перенимают другие.

Уже пользуетесь Claude Code?
```
