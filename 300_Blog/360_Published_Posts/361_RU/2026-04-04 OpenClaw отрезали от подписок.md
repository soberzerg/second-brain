# OpenClaw отрезали от подписок

## Версия для Threads (RU)

### Пост 1

Вчера Anthropic отрезал OpenClaw от подписок.

Люди в панике отменяют подписки. А я спокойно работаю дальше, потому что изначально настроил всё по-другому.

### Пост 2

Большинство людей прокидывали подписку Claude прямо внутрь OpenClaw. Один инструмент, одна подписка, всё в одном месте. Удобно, пока работает.

Я сделал иначе: Claude Code у меня работает отдельным процессом, по своей подписке. А OpenClaw рядом, на GPT-5.4. Но зачем так?

### Пост 3

OpenClaw у меня как менеджер: принимает задачу, разбивает на шаги, контролирует результат. А Claude Code рядом пишет код, прогоняет тесты, деплоит.

Два процесса, два инструмента, каждый на своей модели. За две недели так навайбкодил 3 приложения, не прикасаясь к коду руками.

### Пост 4

Вчерашний твит ничего в этой схеме не сломал. Anthropic отрезал подписку внутри сторонних инструментов, а мой Claude Code работает сам по себе, напрямую.

Если бы я прокинул подписку через OpenClaw как все, сегодня бы тоже искал выход.

А вы уже ощутили пользу от Claude Code, работающего прямо на сервере?
