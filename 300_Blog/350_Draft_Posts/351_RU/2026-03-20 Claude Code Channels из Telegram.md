# Claude Code Channels из Telegram

**Платформа:** Telegram
**Статус:** Черновик
**Дата:** 2026-03-20

---

Увидел в дайджесте Claude Code Channels, настроил за полчаса. Первая команда агенту — из очереди в магазине.

Anthropic сегодня выкатили управление Claude Code через Telegram и Discord. По сути, MCP-канал: пишешь в бот, агент выполняет в терминале. Без ноутбука, без SSH, без VPN.

Настроил по инструкции, завелось быстро. И сразу дал агенту первую боевую задачу: сделать функционал субагентов в AGIents.

Но вот что интересно. Больше всего меня волновало, как перенесётся в Telegram режим планирования. В Claude Code есть механизм, когда агент показывает план и ждёт твоего апрува перед выполнением. В терминале это работает идеально. В мессенджере — пока нет. Что-то ломается на этапе подтверждения.

Короче, управлять агентом с телефона — уже реальность. Но UX ещё сырой. Планирование и апрув — это как раз те штуки, которые отличают "ассистента" от "автопилота". И пока они не работают в мобильном режиме, это скорее удалённый терминал, чем полноценный remote agent.

Буду ковырять дальше. Когда заработает планирование — покажу полный флоу.

А вы уже пробовали управлять кодинг-агентом со смартфона? 👇
