```txt
#вайбкодинг лайфхак №2

В прошлый раз я рассказал, что Claude Code — лучший инструмент для вайб-кодинга. Но многие подумали: «Командная строка? Серьёзно? Я же не разработчик».

Хорошая новость: командная строка не обязательна.

Claude Code уже встроен в десктопное приложение Claude — наравне с Claude Cowork. Открываете Claude Desktop, выбираете Claude Code — и работаете. Без терминала, без настроек.

А если вы уже привыкли к IDE — Claude Code работает и там:
• Cursor — в терминале
• VS Code — через официальное расширение

В крайнем случае, вы можете смотреть файлы проекта в IDE, а работать с агентом в Claude Desktop.

То есть вы получаете тот самый лучший агент для кодинга — но в привычной среде.
```
