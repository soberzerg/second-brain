## Слайды (10)

### Слайд 1 — Cover / Hook
- **Роль:** cover / hook
- **Заголовок:** 7 Claude Code skills, которые я использую каждый день
- **Ключевое сообщение:** Skills — это не «ещё одна фича Claude», а операционная система рабочего дня. Вот 7 штук, без которых я больше не открываю редактор.

### Слайд 2 — Skill 1
- **Роль:** контент / concept
- **Заголовок:** `/brainstorming` — мышление, а не код
- **Ключевое сообщение:** Запускаю в начале любой задачи: skill заставляет распаковать проблему вслух до первой строчки кода. Экономит часы переделок и спасает от «начал писать — понял, что не то».

### Слайд 3 — Skill 2
- **Роль:** контент / concept
- **Заголовок:** `/plan` — план, который Claude сам и выполнит
- **Ключевое сообщение:** Skill превращает идею в пошаговый план с критериями приёмки. Главное — план остаётся в той же сессии: между «придумать» и «сделать» не теряется ни одного нюанса.

### Слайд 4 — Skill 3
- **Роль:** контент / concept
- **Заголовок:** `/tdd` — тесты раньше реализации
- **Ключевое сообщение:** Skill жёстко блокирует код, пока не написаны тесты. Звучит душно, но именно это разгоняет агентов: красный тест — чёткий критерий «готово», без него агент крутится по кругу.

### Слайд 5 — Skill 4
- **Роль:** контент / example
- **Заголовок:** `/ralph-loop` — кодинг, пока ты спишь
- **Ключевое сообщение:** Skill запускает агента в цикле против plan-документа: чинит, тестит, коммитит, повторяет. Утром забираешь результат, а не «прогресс на 60%».

### Слайд 6 — Skill 5
- **Роль:** контент / concept
- **Заголовок:** `/systematic-debugging` — баги без угадайки
- **Ключевое сообщение:** Skill ведёт по протоколу: воспроизведи → изолируй → сформулируй гипотезу → провер тестом. Перестаёшь «фиксить наугад» в три ночи и тратить контекст на петли.

### Слайд 7 — Skill 6
- **Роль:** контент / example
- **Заголовок:** `/ingest` — Inbox → база знаний за минуту
- **Ключевое сообщение:** Кидаю заметку в `000_Inbox/`, запускаю skill — он читает, классифицирует, кладёт в нужный раздел вики с правильным frontmatter. Inbox перестаёт зарастать сам.

### Слайд 8 — Skill 7
- **Роль:** контент / example
- **Заголовок:** `/social-content` — пост под платформу одной командой
- **Ключевое сообщение:** Skill превращает черновик или голосовую заметку в пост под Threads / Telegram / X — с правильным хуком и платформенным стилем. Личный контент-бот, который не теряет твой голос.

### Слайд 9 — Bonus / Framework
- **Роль:** framework / synthesis
- **Заголовок:** Связка дня: brainstorm → plan → tdd → ralph-loop
- **Ключевое сообщение:** Первые четыре skills — это один pipeline: подумал → распланировал → запустил цикл → получил результат. Остальные три (`debugging`, `ingest`, `social-content`) — горизонтальные ускорители, которые подключаются по ходу.

### Слайд 10 — CTA
- **Роль:** CTA
- **Заголовок:** Сохрани и забери мой CLAUDE.md
- **Ключевое сообщение:** Сохрани карусель — пройдись по skills сегодня. Подпишись на @AISobolev: в закрепе мой реальный CLAUDE.md, под который эти skills и заточены.
