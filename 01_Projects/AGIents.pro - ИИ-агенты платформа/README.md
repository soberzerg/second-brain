# AGIents.pro - Платформа для создания ИИ-агентов

## Описание проекта
B2B no-code платформа для создания ИИ-агентов

## Проблема
Решает задачу по созданию кастомных агентов для корпоративных клиентов без необходимости программирования

## Целевая аудитория
- B2B клиенты
- Компании, которым нужны кастомные ИИ-агенты
- Бизнесы без технических ресурсов для разработки

## Ключевые особенности
- No-code подход
- Кастомизация под клиента
- B2B модель

## Связанные проекты
- **AGIency.pro** - агентство, которое использует эту платформу для реализации автоматизаций клиентам

## Статус
Активная разработка

## Связанные папки
- `Research/` - исследования и материалы
- `Chats/` - диалоги с Claude по проекту
- `Daily Progress/` - ежедневный прогресс
