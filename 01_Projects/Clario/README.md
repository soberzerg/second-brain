# Clario - Платформа ИИ-аналитики продаж

## Описание проекта
B2B платформа для аналитики продаж с использованием ИИ

## Моя роль
**CTO-кофаундер**
- Собрал и руковожу технической командой
- Разработка архитектуры системы
- DevOps и процессы разработки
- Стратегическое техническое развитие

## Тип бизнеса
- B2B продажи
- Отдельная компания (не связана с AGIents)
- Есть техническая команда

## Направления работы
- Аналитика продаж
- ИИ-инсайты для B2B команд
- Платформенное решение

## Статус
Активная разработка с командой

## Связанные папки
- `Chats/` - диалоги с Claude по проекту
- `Daily Progress/` - ежедневный прогресс

## Заметки
- Отдельный проект от AGIents/AGIency
- Фокус на B2B сегменте
- Командная разработка vs соло в AGIents
