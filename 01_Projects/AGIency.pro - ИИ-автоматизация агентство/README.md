# AGIency.pro - Агентство ИИ-автоматизации

## Описание проекта
Агентство, специализирующееся на автоматизации бизнес-процессов с помощью ИИ-агентов

## Бизнес-модель
- Консалтинговые услуги
- Реализация автоматизаций для клиентов
- Использует платформу AGIents.pro как основной инструмент

## Услуги
- Создание кастомных ИИ-агентов для клиентов
- Автоматизация бизнес-процессов
- Консалтинг по внедрению ИИ

## Связанные проекты
- **AGIents.pro** - собственная платформа для создания агентов
- Все автоматизации реализуются на этой платформе

## Статус
Активная работа

## Связанные папки
- `Research/` - исследования рынка и кейсов
- `Chats/` - диалоги с Claude по проекту
- `Daily Progress/` - ежедневный прогресс
