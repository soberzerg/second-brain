## Фаза 1: Стабилизация

### Цель

Устранить критические уязвимости безопасности, предотвратить потенциальные инциденты.

### Работы (предварительная оценка объема)

| Наименование работ, результат | Объем, часов |
| :---- | :---- |
| Опросник, планирование, организационные вопросы | 4 |
| Аудит безопасности и отчёт по explorer-service | 4 |
| Аудит безопасности и отчёт по другим сервисам (\~5-8 сервисов) | 20 |
| Аудит инфраструктуры — инфраструктурная диаграмма (актуализация) | 4 |
| Рекомендации по внедрению базового логирования security-событий | 4 |
| Рекомендации по настройке алертов на подозрительную активность | 4 |
| Презентация, обучение команды принципам безопасной разработки \+ руководство | 6 |
| Презентация, обучение команды разработке с использованием AI \+ руководство | 4 |
| Консультирование команды по реализации рекомендаций | 6 |
